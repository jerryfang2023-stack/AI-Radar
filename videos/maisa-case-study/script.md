# 《Maisa：让不会编程的人自己建 AI》案例视频脚本

- 画幅：1920×1080，16:9
- 语种：中文普通话
- 旁白：克制、有冲击力的商业案例男声；短句、反问、数据重音
- 目标时长：约 2–3 分钟（以生成音频为准）
- 叙事：融资钩子 → 失败追问 → 具体冲突 → 产品解法 → 工作场景 → 审计机制 → 客户结果 → 组织价值 → 行动结尾

旁白与画面标题的单一数据源为 [`src/scenes.json`](src/scenes.json)。视频对原文做了两处编辑处理：

1. 参考“结果先行、反常识解释、场景拆解、价值升华”的案例叙事，但不复制参考视频原句；
2. 将“2500 万美元 A 轮”纠正为 Maisa 与 Creandum 官方披露的“2500 万美元种子轮”；
3. 将客户效果明确表述为 Maisa 公司披露，且提示客户名称未公开、指标仍需独立验证。
