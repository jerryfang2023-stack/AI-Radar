# Legacy Import Inbox

This folder receives V1 and recent monitoring assets that are valuable but not yet transformed into V2 production assets.

Do not copy old V1 Markdown here as final content. Each item must first be classified in:

```text
agent-workflow/reports/WSD-20260507-18-legacy-content-inventory.md
agent-workflow/reports/WSD-20260507-18-v2-content-migration-map.md
```

Assets can move downstream only after they have:

- Stable V2 ID.
- Original source path.
- Original date.
- Conversion date.
- Conversion reason.
- V2 destination.
- Relationship fields.
- Evidence gaps or uncertainty notes.
