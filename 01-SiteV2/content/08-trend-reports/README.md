# 08-trend-reports｜趋势追踪报告

本目录保存趋势快报、深度趋势报告和不成稿决定。

```text
flash/               -> 趋势快报
full/                -> 深度趋势报告
no-report-decisions/ -> 证据不足、不成稿或延期说明
```

趋势追踪不是卡片聚合页。报告必须说明底层变化、场景 / 案例、观点参照、来源类型、证据缺口、反证和观察窗口。

新报告只写入 `flash/`、`full/` 或 `no-report-decisions/`，不写入根目录。
