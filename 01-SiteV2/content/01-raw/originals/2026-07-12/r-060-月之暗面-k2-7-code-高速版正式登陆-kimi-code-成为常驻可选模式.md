# 月之暗面 K2.7 Code 高速版正式登陆 Kimi Code，成为常驻可选模式

## clean_text

IT之家 7 月 11 日消息，月之暗面官方昨晚宣布， K2.7 Code 高速版结束 Beta，成为常驻可选模式 。
订阅 Allegretto 及以上会员计划的用户，在 Kimi Code CLI 或其他工具中使用 Coding Plan 进行编程时，无需任何申请，可以直接调用 K2.7 Code HighSpeed。
官方表示，K2.7 Code HighSpeed 在 Coding Plan 中的用量消耗是普通版模型的 3 倍 。用户可以根据实际任务需求和当前用量，选择是否开启高速模式。
据介绍，K2.7 Code 高速版和普通版使用同一个模型， 输出速度约为普通版的 5-6 倍 。常规编程场景下，取输入长度中位数，输出速度约 180 Token/s；短上下文场景可达 260 Token/s。
Kimi K2.7 Code 高速版价格为 Kimi K2.7 Code 的两倍： 1M tokens 的标准输入和输出价格分别为 13 元和 54 元；命中缓存的输入价格为 2.6 元 。
据IT之家此前报道， Kimi K2.7 Code 编程模型发布于 6 月 12 日 。内外部基准评估显示，Kimi K2.7 Code 相比 K2.6 模型显著提升了长上下文编程场景的指令遵循能力、长程编程任务的性能表现，并且大幅改善了在长程任务中的过度思考倾向，平均 token 消耗减少 30%。
