IT之家 7 月 19 日消息，阿里巴巴今天宣布，开放式世界模型 HappyOyster 1.0（快乐生蚝）现已登陆阿里云百炼，面向企业和开发者开启灰测。
IT之家了解到，HappyOyster 1.0 版本围绕「用户与世界的关系」，提供世界探索（Adventure）与实时导演（Directing）两大核心模式。
其中， 世界探索模式支持用户输入一句话或一张图 ， 生成可实时探索和交互的开放世界 。模型能够根据用户输入创建任意角色在任意世界中探索，根据实时交互指令，预测世界的下一步发展，给出响应，支持 1 分钟以上的主体动作和环境交互体验，音画同出。
而实时导演模式可让用户像拍摄电影一样，用文字指令实时操控镜头、调度角色、改变剧情走向、画面即时响应、持续演绎；你随时可以暂停、改写、回溯。当前模型能力在数字人互动陪伴、剧情与舞台演绎、 POV 沉浸式模拟体验等场景中表现优异，具备较强的实时互动性。
此外，HappyOyster 1.0 提供 Android / iOS / Web 三端 SDK。开发者服务端通过 Open API 管理世界，客户端 SDK 封装了 RTC 连接、视频渲染和实时交互，支持 Adventure 与 Directing 两种模式，体验结束后可导出视频。产品支持企业用户和开发者通过 SDK 快速构建可玩世界、互动剧、AI 陪伴等沉浸式应用。
