# 蚂蚁集团副总裁陈亮：智能体越强，信任风险越不容忽视

## clean_text

IT之家 7 月 19 日消息，2026 世界人工智能大会今天在上海举行。据新浪科技消息，蚂蚁集团副总裁、蚂蚁大安全 CTO 陈亮认为， 智能体（Agent）正在成为 AI 时代的新型操作系统 ， 需要一套与之相匹配的信任基座 。
据悉，蚂蚁集团为此提出了“原生安全”和“可信互联”两个方向的智能体信任基础。前者拥有不同终端形态的 Agent 运行底座，包括面向智能眼镜、耳机等轻量设备的可穿戴设备 Agent 操作系统 LingDevice OS，以及面向多用户、多任务复杂业务场景的企业级 Agent 操作系统。
其中， 可穿戴设备 Agent 操作系统强调端云协同 、 低功耗和跨设备连接 ；企业级 Agent 则面向复杂业务环境，为多智能体任务提供高并发、分布式的调度能力。
同时，蚂蚁针对智能体任务的不确定性提出原生安全和“换挡式”编排调度机制，通过分布式运行底座兼顾执行效率、推理能力与运行成本，并将安全能力进一步下沉到操作系统层。
互联网公开资料显示，陈亮花名俊义，早期在蚂蚁集团负责“ 双十一”支付技术保障与系统稳定性工作 。此后组建蚂蚁集团 SRE 团队，负责 AI 基础设施、数据库等技术研发。他曾任支付宝事业群 CTO，期间带领团队参与 AI 大模型技术迭代，并推动 AI 技术在支付宝业务中的应用 。
