感谢IT之家网友 GreatMOLA 、 Domado 的线索投递！
IT之家 7 月 19 日消息，阿里云官方今日宣布， 阿里 Token Plan 个人版正式发布 ，用户可在其中抢先体验 Qwen3.8-Max-Preview。
官方表示，Qwen3.8-Max 参数量达 2.4T，在代码工程和专业办公等场景中表现出色，可能是除了 Fable 5 外最强大的模型，正式版将于近期发布并开源。
IT之家附阿里 Token Plan 个人版价格如下：
Lite
39 元 / 月（6.5 折，原价 60 元 / 月）
2,500 Credits / 7 天
700 Credits / 5 小时
可同时支持 1～2 个 Agent 并发运行
Standard
139 元 / 月（7.7 折，原价 180 元 / 月）
10,000 Credits / 7 天
3,000 Credits / 5 小时
可同时支持 3～4 个 Agent 并发运行
Pro
499 元 / 月（8.3 折，原价 600 元 / 月）
40,000 Credits / 7 天
12,000 Credits / 5 小时
可同时支持 6～8 个 Agent 并发运行
官方还推出了 Qwen3.8-Max-Preview 限时活动：白天 Credits 消耗 1 折，个人版 Token Plan 夜间（每日 22:00 至次日 08:00）在此基础上再享 2 折（即 0.2 折）。
除 Token Plan 外，Qoder 和 QoderWork 也已首发支持 Qwen3.8-Max-Preview，千问 PC 端可免费体验。
Token Plan 体验地址如下：
https://platform.qianwenai.com/pricing/token-plan
