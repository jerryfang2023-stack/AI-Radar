New Check Inventory by Resource Type API available鈥擟heck Inventory API retiring September 25, 2026
A new Check Inventory by Resource Type API is now available. The existing Check Inventory API will be retired on September 25, 2026.
Date : June 23, 2026
Workspace : APIs
Impacted audience : All partners using Partner Center APIs for inventory checks.
A new Check Inventory by Resource Type API is now available, enabling partners to check inventory for catalog items filtered by a specific resource type. This API replaces the existing Check Inventory API, which will be retired on September 25, 2026 .
What's new
The new Check Inventory by Resource Type API was introduced to handle the growing complexity of Azure services, where availability depends not just on country but also on resource type. By requiring a resourceType parameter, it enables more scalable and performant inventory checks to address data growth challenges without changing the API response contract. The API filters inventory results by resource type (for example, VirtualMachines , SqlDatabases , CosmosDb ).
What's changing
The existing Check Inventory API ( /v1/extensions/product/checkinventory ) will be retired on September 25, 2026 .
Partners must migrate to the new Check Inventory by Resource Type API ( /v1/extensions/product/checkinventorybyresourcetype ) before the retirement date.
Next steps
Review the Check Inventory by Resource Type API documentation for supported resource types and usage examples.
Update your integrations to use the new API before September 25, 2026 .
