# Claude Design 上线首周用户破百万，强化与 Claude Code 双向联动

## clean_text

IT之家 6 月 18 日消息，Anthropic 今天（6 月 18 日）发布公告，在宣布 Claude Design 上线 首周用户规模突破 100 万外， 进一步强化和 Claude Code 的双向联动，实现从设计到编程的无缝工作流。
Anthropic 表示重构设计系统导入功能，Claude Design 现支持从 GitHub 仓库、设计文件或直接上传素材中获取一个或多个设计系统，自动检测输出内容是否符合现有组件标准。
Anthropic 还进一步强化 Claude Design 与 Claude Code 的协同工作，用户可使用 /design-sync 命令拉取设计系统，让 Claude Design 基于现有组件开始制作。IT之家附上相关截图如下：
用户完成设计后支持直接传递给 Claude Code，继承先前工作并转化为软件，从 Claude Code 端也可通过“/design”命令创建、编辑、同步设计项目。
编辑器方面，新版提供直接、精细的控制，新增布局控件支持拖拽、调整大小和对齐元素，数百项修复提高稳定性。
使用配额方面，Claude Design 现在和 Chat、Claude Cowork 和 Claude Code 共享使用限制，大多数用户将获得更多创作空间，平均每次交互消耗更少的 Token，错误率显著下降。
导出方面，Claude Design 支持可靠导出为 PDF 和 PowerPoint，或发送至 Adobe、Base44、Canva、Gamma、Lovable、Miro、Replit、Vercel 和 Wix 等应用。
