# 英伟达 ACE Game Agent SDK 进入 Beta 测试，内置 Qwen 3.5 4B 模型并支持 UE5

## clean_text

IT之家 6 月 18 日消息，在昨日开幕的虚幻嘉年华 2026 期间，英伟达宣布其用于创建和构建 AI 游戏队友的 NVIDIA ACE Game Agent SDK 现已进入 Beta 测试阶段。
这套工具允许游戏开发者为搭载 GeForce RTX 显卡的设备构建本地运行的 AI 角色，并包含面向虚幻引擎 5 的插件，覆盖 AI 驱动的自动语音识别、小语言模型和文本转语音组件。
该技术此前已有实际应用案例。KRAFTON 近期在《绝地求生：大逃杀》中推出了限时 Beta 测试的 AI 队友 Ally，用于测试和收集玩家对“人机双排模式”的体验反馈。IT之家注意到，另一个开发中的案例是 Creative Assembly 在《全面战争：法老》中加入的 AI 顾问。
据英伟达官方介绍，NVIDIA ACE Game Agent SDK 被设计为一个轻量级的 C/C++ 智能体框架，内置即用型 Qwen 3.5 4B 模型用于决策，以及超轻量 Chatterbox Turbo 350M 模型用于语音合成。
正如 PUBG Ally 所展示的，ACE 游戏智能体可以在仅配备 8GB 显存的 GPU 上运行，包括 GeForce RTX 3060 等中端显卡均可部署。
其 SDK 包含三类核心 API。Agent API 负责管理聊天历史并在极少代码量下实现多步推理，Chat API 让开发者直接控制推理过程，RAG API 则聚焦游戏特定知识，利用开发者提供的数据库和最新信息来检索并回答玩家问题、执行行动。
借助虚幻引擎 5 插件，所有这些工具和功能均可存储在游戏内部，然后在本地 RTX 硬件上运行，无需任何云服务器的辅助。
除此之外，英伟达还宣布了更多后续计划，包括与 Epic 合作将 NVIDIA ACE 引入虚幻引擎的 MetaHuman NPC 技术，利用模型进行语音合成，并结合 Audio2Face 技术实现考虑角色情感和性格特征的面部动画。
与此同时，NVIDIA 发布了新的 DLSS 4.5 虚幻引擎插件，增加了对 DLSS 4.5 动态多帧生成、6X 模式多帧生成以及第二代 Transformer 超分模型的支持。NVIDIA Kimodo 这一用于可提示、可控人类动作生成的开源项目也已上线，其配套插件 Animotive Kimodo 可直接在虚幻引擎中使用，Blender 和 Maya 版本将随后推出。

## 原始摘要 / 采集文本

英伟达在虚幻嘉年华 2026 宣布 NVIDIA ACE Game Agent SDK 进入 Beta 测试。该轻量级 C/C++ 框架内置 Qwen 3.5 4B 模型用于决策、Chatterbox Turbo 350M 用于语音合成，包含 UE5 插件，可在 8GB 显存 GPU 上本地运行。SDK 提供 Agent API、Chat API 和 RAG API。已有 KRAFTON 在《绝地求生》推出限时 AI 队友 Ally，Creative Assembly 在《全面战争：法老》加入 AI 顾问。同期英伟达还发布 DLSS 4.5 UE5 插件（支持动态多帧生成、6X 模式、第二代 Transformer 超分）及开源项目 NVIDIA Kimodo 与 Animotive Kimodo 插件。
