[06] FAQ
Need Answers?
How does TesterArmy test my website? TesterArmy launches a real browser, navigates your site like a human, and uses AI to evaluate the results. You describe what to test in plain language, and the agent handles clicking, typing, and verifying.
Is there a free trial? Yes, every account starts with a free trial so you can evaluate TesterArmy on real projects before upgrading.
How is this different from Playwright, Cypress, or Playwright MCP? Different layer. Playwright and Cypress are frameworks - you write and maintain the tests, debug the flake, manage the selectors. Playwright MCP gives an agent raw browser control. TesterArmy is a service - you describe the journey in plain English, we run it, maintain it, and ship evidence on every release. Same browser primitives underneath, no test code in your repo.
Can I run tests in CI? TesterArmy integrates with GitHub Actions, GitLab CI, and any pipeline that can hit a webhook. You get check statuses and report links right on your pull requests.
Can it handle authentication and OTPs? Yes. The agent logs in with stored credentials, handles OAuth, and receives one-time passwords via dedicated per-agent inboxes. Credentials stay encrypted with AES-256-GCM at rest.
Do you support mobile apps? Yes, TesterArmy tests iOS and Android builds alongside web and mobile-web experiences from the same project.
