# OpenAI 更新 Mac 端 ChatGPT 应用，整合 Chat、Work 和 Codex 模式

## clean_text

IT之家 7 月 18 日消息，OpenAI 核心产品负责人蒂博 · 索蒂奥（Tibo Sottiaux）昨日（7 月 17 日）在 X 平台发布推文， 分享了一组新截图，展示了适用于苹果 macOS 的全新 ChatGPT 应用。
OpenAI 于 7 月 9 日宣布推出全新桌面应用程序， 整合 Codex 和 ChatGPT 体验 ，增强 AI 编程体验的同时，扩展覆盖更多非开发者群体。
桌面版 Codex 在更新后，将会变成全新的 ChatGPT 桌面应用，包括 Chat（聊天）、ChatGPT Work（工作）和 Codex 三个组成部分，更新后应用会保留现有的 Codex 任务和项目，也可以在设置中选择 Codex 应用图标。
IT之家翻译其推文内容如下：
我们收到了很多关于新 ChatGPT 桌面应用的宝贵反馈（我们第一次尝试后并没有完全做到完美），因此我们进行了一些调整。
用户可以在侧边栏中查看 ChatGPT 的对话历史和项目。用户的 Chat 和 Work 历史现在可以在网页版、移动版和桌面版之间同步。本地任务仍然保存在你的电脑上。
用户现在可以在桌面版的 ChatGPT 中轻松切换 Chat 和 Work 模式，显示方式与网页版、移动版保持一致。
Codex 模式的用户的体验没有任何变化。它仍然是 OG，在它擅长的领域里是最棒的。
总体来说，我们将继续修复小问题并提升性能、可靠性和效率。
