# Raw Originals

本目录保存日常监测 v2 的 Raw 本地原文档案。

每条进入 `06-content/01-raw/YYYY-MM-DD-raw-candidates.md` 的 Raw 线索，都必须对应一个本地原文文件：

```text
06-content/01-raw/originals/YYYY-MM-DD/R-001-source-title.md
```

原文档案用于内部研究、复核、二次搜索和深挖证据链，不直接进入正式前台。

## 文件模板

```markdown
---
raw_id:
source_name:
source_url:
source_type:
captured_at:
language:
copyright_note: local research archive only
---

# 原文标题

## 原文 / 可用正文

保存网页正文、官方公告正文、社媒原帖文本、播客转写片段、招聘 JD 正文或合法可保存摘录。

## 中文摘录 / 机器翻译

可选。

## 采集备注

说明是否为一手来源、是否需要二次搜索、是否存在可信度问题。
```
