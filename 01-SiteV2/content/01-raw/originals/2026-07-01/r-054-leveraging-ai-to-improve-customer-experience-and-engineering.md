I completely overhauled how I build with AI and honestly it changed everything.
Here's what I figured out:
1. Short prompts win:
I used to write these massive detailed prompts. Now almost everything I send is 1-2 sentences. The model is smart enough, you just have to trust it.
2. Fresh thread for every task:
Old context is the enemy. Every new task gets a clean slate so the model isn't biased by whatever I was doing before.
3. Stop telling it HOW. Tell it WHAT:
I give the model the goal, not the implementation. Half the time it finds a better way to do it than I would have anyway.
4. Read what the model says, not just the code:
This was the biggest unlock for me. Most people skim the text and jump to the code. That's backwards. The conversation is where the real value is.
5. Remote coding is underrated:
Running agents on a separate machine while I do other things has been a game changer. No more babysitting a running process.
6. The pattern I keep coming back to:
Keep it simple. The more I tried to engineer my workflow, the worse it got. The moment I stripped it back, productivity went through the roof.
If you're still copy-pasting errors into chat and micromanaging every file, try letting go a little. You might be surprised.

I completely overhauled how I build with AI and honestly it changed everything.
Here's what I figured out:
1. Short prompts win:
I used to write these massive detailed prompts. Now almost everything I send is 1-2 sentences. The model is smart enough, you just have to trust it.
2. Fresh thread for every task:
Old context is the enemy. Every new task gets a clean slate so the model isn't biased by whatever I was doing before.
3. Stop telling it HOW. Tell it WHAT:
I give the model the goal, not the implementation. Half the time it finds a better way to do it than I would have anyway.
4. Read what the model says, not just the code:
This was the biggest unlock for me. Most people skim the text and jump to the code. That's backwards. The conversation is where the real value is.
5. Remote coding is underrated:
Running agents on a separate machine while I do other things has been a game changer. No more babysitting a running process.
6. The pattern I keep coming back to:
Keep it simple. The more I tried to engineer my workflow, the worse it got. The moment I stripped it back, productivity went through the roof.
If you're still copy-pasting errors into chat and micromanaging every file, try letting go a little. You might be surprised.
