Coasty, a new startup from YC S26, has introduced an API designed for creating computer-use agents. This development aims to automate complex digital tasks, with potential implications for productivity and AI integration.
Coasty, a startup from YC S26, has officially launched its API to enable the development of computer-use agents. The platform aims to facilitate automation of complex digital tasks, marking a step toward more autonomous AI-driven software tools. This release is significant for developers and companies exploring automation and AI integration in their workflows.
Founded by Nitish and Prateek, Coasty’s API provides a framework for building agents capable of performing various computer-based tasks, such as data management, web automation, and digital interactions. The company described the platform as a way to ’empower developers to create intelligent agents that can operate across different software environments.’
The API is now publicly accessible, with early adopters already testing its capabilities. Coasty emphasizes that their agents are designed for flexibility, enabling automation in both personal and enterprise contexts. The startup has shared that their goal is to reduce manual effort and increase productivity through AI-driven automation tools.
