# Puter Labs 上线 Firefox in WebAssembly 实验项目，可在 Chrome 中运行完整火狐浏览器

## clean_text

IT之家 7 月 16 日消息，Puter Labs 上线 Firefox in WebAssembly 实验项目，可以在 Chrome 浏览器中启动完整火狐 Firefox 浏览器， 将完整的浏览器内核（Gecko）编译成了 WebAssembly (Wasm)。
IT之家注：WebAssembly 是一种在现代浏览器中运行的低级、类汇编的二进制代码格式。它支持开发者用 C、C++、Rust 等语言编写高性能应用，并以接近原生硬件的速度在网页端执行。
内核方面，Firefox in WebAssembly 项目利用 Emscripten 工具链，将原本由 C++ 编写的 Firefox 渲染引擎成功编译为 Wasm。
此外为了提升执行速度，该项目把 JavaScript 执行引擎 SpiderMonkey 编译为 WebAssembly，并加入实验性 JIT（即时编译）编译器，并支持基于 WebGL 的 GPU 加速。
项目说明显示，网页加载完成后，用户点击“Launch Firefox”即可在浏览器内启动 Firefox。
开发成本方面，团队使用了 Claude Opus、Claude Fable 等 AI 模型，累计消耗约 300 亿个 token，费用超过 25000 美元（现汇率约合 16.9 万元人民币）。
