来源： IT之家
作者： 故渊
责编： 故渊
评论：
IT之家 7 月 3 日消息，科技媒体 Windows Latest 昨日（7 月 2 日）发布博文，分享了一段视频， 展示了微软在 2024 年（可能更早）开发的 Copilot 系统，内部代号为 Project Aion。
底层方面，该系统基于 Microsoft Edge 浏览器和 Web3 轻量化 Windows 代码库构建，是完全针对 AI 场景设计开发的系统，没有传统开始菜单和桌面图标，一切均基于 AI 交互实现。IT之家附上演示视频如下：
界面方面，用户启动 Project Aion 系统后，在底部会有一个任务栏，点击后会启动 Copilot。在打开窗口后，会包含“Stay on Top”（展示 Microsoft 365 用户信息流）、“Create something new”（创造新事物）和“News of the day”（今日要闻）。
任务栏加入 Spaces（空间）功能，可自动把已打开应用和网站归入不同区块。这些 Spaces 也出现在开始菜单中，用户可通过单次点击打开一组标签页和应用集合。
应用方面，该系统不支持传统 Windows 原生应用，只支持 Web 应用以及访问网站。如需桌面应用，微软在视频中建议使用 Windows 365。
广告声明：文内含有的对外跳转链接（包括不限于超链接、二维码、口令等形式），用于传递更多信息，节省甄选时间，结果仅供参考，IT之家所有文章均包含本声明。

## 原始摘要 / 采集文本

微软内部代号 Project Aion 的系统基于 Microsoft Edge 浏览器和 Web3 轻量化 Windows 代码库，完全面向 AI 场景，无传统开始菜单和桌面图标，一切通过 AI 交互。启动后底部任务栏可启动 Copilot，窗口含"Stay on Top"（展示 Microsoft 365 用户信息流）、"Create something new"和"News of the day"。任务栏 Spaces 功能自动将已打开应用和网站归入不同区块，可通过单次点击打开一组标签页和应用集合。该系统不支持传统 Windows 原生应用，只支持 Web 应用和网站，如需桌面应用则建议使用 Windows 365。
