SwitchBoard AI
Autonomous procurement and compliance layer for AI agent marketplaces.
SwitchBoard AI is an always-on buyer-operator and compliance layer that discovers marketplace sellers, runs paid micro-trials, scores output quality and trust signals, selects vendors, purchases finalized outputs, assembles buyer-ready deliverables, and exposes a human-readable audit trail - all autonomously.
Built in 22 hours at the Nevermined Autonomous Business Hackathon (March 2026, AWS Builder Loft, San Francisco).
Live: switchboardai.ayushojha.com
How It Works
Client request
ProcurePilot (buyer engine)
|--- discovers vendors in Nevermined marketplace
|--- runs paid micro-trials via x402 tokens
|--- scores quality, latency, trust
|--- selects winner (or falls back to in-house)
|--- assembles final deliverable
Delivered brief + full audit trail
Every step is metered, logged, and visible in the dashboard.
Architecture
SwitchBoard AI runs as an autonomous procurement and compliance layer with a 7-agent execution flow:
Discovery Agent finds candidate sellers and marketplace categories
Purchaser Agent acquires plans and runs paid vendor trials
Scorer Agent evaluates quality, freshness, latency, and cost-efficiency
Switcher Agent compares outcomes and selects the best supplier stack
Assembler Agent composes the final buyer-ready packet
DataForge Web executes structured extraction tasks
DataForge Search executes semantic research tasks
TrustScore sits across the flow as the governance and compliance overlay, while the dashboard exposes the human-readable audit layer for judges and operators.

Local Development
# Clone
git clone https://github.com/Ayush10/switchboard-ai.git
cd switchboard-ai
# Create .env from example
cp env.production.example .env
# Fill in your API keys (NVM_API_KEY, OPENAI_API_KEY, etc.)
# Build and run
docker build -t switchboard-ai .
docker run --env-file .env -p 8000:8000 -v $( pwd ) /storage:/app/storage switchboard-ai
The app will be available at http://localhost:8000 .
