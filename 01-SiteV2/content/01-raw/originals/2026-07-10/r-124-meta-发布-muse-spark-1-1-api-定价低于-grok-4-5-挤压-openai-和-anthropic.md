# Meta 发布 Muse Spark 1.1，API 定价低于 Grok 4.5，挤压 OpenAI 和 Anthropic

## clean_text

Meta 推出多模态推理模型 Muse Spark 1.1，专为智能体任务、编程和计算机使用设计，支持多智能体编排与百万 token 上下文窗口。其 API 定价为输入 1.25 美元、输出 4.25 美元、缓存 0.15 美元（每百万 token），低于昨日发布的 xAI Grok 4.5，且远低于 OpenAI 和 Anthropic 竞品。该模型在 VALS-AI 榜单排名第四，Vibe Code Bench 编程评测较前代跃升 36 位。Muse Spark 1.1 为闭源模型，首次通过 Meta Model API 公开预览，但 Muse Image 尚未开放。Meta 以低价策略对依赖高利润率的 AI 实验室形成压力，同时面临中国开源模型的低价竞争。
