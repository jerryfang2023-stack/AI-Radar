# Google AI Studio 为 Build 模式新增"从 GitHub 导入"功能

## clean_text

Google AI Studio 在 Build 模式中推出"import from GitHub"功能。用户指向一个 GitHub 仓库，AI Studio 将其转换为运行时兼容格式，随后可在 AI Studio 中迭代并部署到 Cloud Run。对于调用 Gemini API 的应用，GEMINI_API_KEY 会被配置为服务器端密钥，不包含在客户端代码中。该功能补全了 Build 模式此前缺失的从 GitHub 导入的路径，支持续接已有代码库、快速上手同事分享的仓库或将脚本转化为完整应用。
