# Building a Control Plane for AI Agents in Production - LinkedIn

## clean_text

AI infrastructure has moved through an interesting arc over the last few years, each phase triggered by a jump in model capability:
AI infrastructure has moved through an interesting arc over the last few years, each phase triggered by a jump in model capability:
MAKE IT USEFUL: Give models the right data and context: prompting, RAG, fine-tuning, all to ground outputs and cut hallucinations.
MAKE IT ACT: As models learned to reason, the stack grew hands: tool calling and orchestration. Composio became the default integration layer for agents, how they connect to and act on the tools they use. As base models commoditize, the harness around the model (tools, orchestration, memory) is becoming the real moat.
MAKE IT TRUSTWORTHY: An agent that acts with real credentials is a liability when it acts wrong. So the frontier moved to control: guardrails, security, compliance, and governance, plus the evals and observability to measure, control, and improve how it performs.
Across all three transitions, one need has turned non-negotiable: a control plane to govern every AI agent in production.
That's what we want to dig into. I'm hosting Yash Sheth, co-founder of Galileo (recently acquired by Cisco), to get concrete about what actually runs this layer: logs, traces, and evals, now that agents are shipping real work.
