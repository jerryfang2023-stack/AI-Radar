# SemiAnalysis 洞察 Token 经济：200 美元 AI 订阅榨出 70 倍用量

## clean_text

IT之家 6 月 13 日消息，研究机构和咨询公司 @SemiAnalysis_ 于 6 月 11 日在 X 平台发布推文，指出 Anthropic 的 Claude Max 与 OpenAI 的 ChatGPT Pro 月费虽然都是 200 美元（现汇率约合 1355 元人民币）， 但若按照 API 用量计算，用户最高可消耗价值 8000 美元和 14000 美元（IT之家注：现汇率约合 94872 元人民币）的 Tokens。
该机构为了比较订阅与接口计费差异，购买了 Anthropic 和 OpenAI 的全部订阅方案，并模拟接近高强度专业用户的真实使用状态，持续执行高负载编码任务，直到触及每周使用上限。
结果显示，若按当前 API（应用程序编程接口）价格换算，月费 200 美元的 Claude Max 20x 方案，最高可获得约值 8000 美元的令牌量；ChatGPT Pro 20x 方案对应的令牌量最高约值 14000 美元。
该机构指出在同样的使用规模下，直接调用 API 的成本要高得多。
部分观点认为，这样的价格体系未必能长期维持，尤其当重度用户持续榨满上限后，模型推理成本、算力供给压力与订阅收入之间的矛盾可能会进一步放大。
