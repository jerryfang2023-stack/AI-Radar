# Geordie Partners with Microsoft on Agent Control Specification

## clean_text

Big new from the Geordie AI team at Microsoft Build today.
Big new from the Geordie AI team at Microsoft Build today. 🛠️
Geordie is deepening our partnership with Microsoft to help shape how the industry governs AI at scale.
As more companies run agents in production, security and AI team are asking the same question: how do we stay in control without building a different governance model for every tool, platform, or vendor we work with?
Microsoft's Agent Control Specification is an answer to that. We're contributing our experience from the field and our mitigation engine Beam 💡 now plugs directly into that ecosystem.
The result for you: policies you author and test in Geordie can be enforced through Beam or carried into any ACS-compatible environment. Less fragmentation and more confidence.
We are excited to keep building this out alongside Microsoft as agent adoption grows. Read more from the release in the comments.
Henry Comfort Josh Kriss Hanah-Marie Darley Benji Weber Daniel Sokol Kevin Magee
