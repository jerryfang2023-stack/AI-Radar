# Work Louder 与 OpenAI 合作推出 Codex Micro 键盘

## clean_text

Work Louder 与 OpenAI 合作推出 kbd-1.0-codex-micro 键盘，专为 Codex 智能体工作流设计。键盘配备 13 个机械按键、摇杆和旋钮，每个 Agent Key 可通过 RGB 灯实时显示智能体状态，摇杆可触发 PR 审查等 Codex 技能。支持蓝牙/USB-C 连接，兼容 Mac 和 Windows，预计 7 月 24 日发货。
