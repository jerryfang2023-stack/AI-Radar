# AI 交互革命：Claude Code 系统提示词精简 80%，示例约束反而遏制表现

## clean_text

IT之家 7 月 3 日消息，科技媒体 The Decoder 昨日（7 月 2 日）发布博文，报道称 Anthropic 表示为迎合 Claude Fable 5 模型， 进一步精简 Claude Code 系统提示词，降幅达到 80%。
Anthropic 技术人员塔里克 · 希希帕尔（Tariq Shihipar）表示，引导 AI 模型方式已发生根本性转变，更多的提示词和示例，已无法带来更好的结果。
他表示：“我们最近发现，包括 Fable 5 在内，新模型要更简洁的系统提示，用户输入的示例往往会限制其表现，事实上此类新模型所掌握的示例，要比用户更具想象力。”
他补充道 Anthropic 现在不再使用“禁止这样做”之类的硬性规则，而是尝试通过上下文来引导 Fable 模型。
他认为 AI 提示词正经历“短-长-短”演变，早期模型需要简短的提示，辅以大量示例和限制性指令，而随着模型理解能力的提高，提示逐渐变长；而现在，提示又开始变短了。IT之家附上相关截图如下：
