# Karminski 设计首个大模型复杂工程能力测试：Kimi-K2.7-Code 完成度 93%

## clean_text

Karminski 设计了首个大模型复杂工程能力测试：给模型 21000 token 需求文档，要求通过 Coding Agent 修改 SillyTavern（酒馆）项目，增加动态数值系统。结果 Kimi-K2.7-Code 需求完成度 93%，接近人肉 vibe coding 的 97.7%；Kimi-K2.6 仅 74.4%；DeepSeek-V4-Pro 只有 68.6%。K2.7 漏需求少得益于全局注意力更强、Agent 能力更完善；K2.6 因 Plan 偏离、DeepSeek 因未使用 Plan Agent 导致遗漏。测试揭示了模型"注意力管状视野"问题--只关注易验证任务，忽略需推理的 web 界面、提示词注入等。Kimi-K2.7-Code 的 20 分提升被认为是国产代码模型的生产力拐点。
