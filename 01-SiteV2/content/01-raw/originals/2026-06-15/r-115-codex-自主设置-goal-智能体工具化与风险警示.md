Smart to have it as a tool for agents. Exactly how I have it built in my orchestrator app.
I even built a little UI for /goal in is something you can try if you want to get better goals that agents follow. Mine your agent sessions, collect goals that performed well, and package those insights/best practices as a skill using an automation.
That skill can then be reused with the /goal tool to set even stronger and more reliable goals. Doesn't get more meta than this.
I am thinking of doing a live session on this if folks are interested.
This is a simple workflow with tons of value and ways to optimize the results of /goal. It turns out that some of the weird behavior of LLM (e.g.g, reward hacking, bias to finish quickly, and other weird shortcuts) creep up a lot when using /goal, so you want to be extremely careful of that.
