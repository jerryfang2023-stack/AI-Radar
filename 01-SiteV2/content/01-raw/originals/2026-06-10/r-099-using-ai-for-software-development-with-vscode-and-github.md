Software Development in the Age of AI: Dev Tools
I've been using VSCode, GitHub, Copilot/Gemini/Codex, Codespaces, and Colab for my AI projects. Below are some findings to share:
* GenAI coding assistants and agents can already code at a level comparable to a senior software engineer and are improving rapidly. It's impressive to delegate several tasks to Copilot agents, which work in parallel and return results in minutes—tasks that might take me hours or even days. They follow engineering best practices, including testing and security scans, without prompting. I review, debug, and modify their pull requests in separate Codespaces before merging. It truly feels like managing a software engineering team. While Gemini and Codex CLI offer similar capabilities, they are not as well integrated with VSCode, GitHub, and Codespaces. It's important to note that these coding agents, like humans, can hallucinate, so a strict pull request process is essential.
* Software development is complex and necessitates a sophisticated CI/CD toolchain. My experience leading infrastructure organizations at major tech companies has highlighted this. The Google SRE book and SWE book effectively describe these challenges. I can see the industry consolidating around a standard set of development tools such as VSCode, GitHub, Codespaces, Colab, and GenAI coding agents, which could improve developer productivity by 10X or more.
* I highly recommend the GitHub Student Pack (https://lnkd.in/gc8WcEm7), Gemini for Students (https://lnkd.in/giWuB3Gy), and Copilot & M365 for Students (https://lnkd.in/gvRnwurD). I think all development and AI tools should be free for students and individual developers: it is smart business practice.
