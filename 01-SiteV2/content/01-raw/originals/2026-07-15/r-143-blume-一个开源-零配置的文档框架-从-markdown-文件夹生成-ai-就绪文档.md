# Blume：一个开源、零配置的文档框架，从 Markdown 文件夹生成 AI 就绪文档

## clean_text

OpenAI 开发者 Hayden Bleasel 发布了 Blume 1.0.3，一个 MIT 许可的开源文档框架。用户只需将 Markdown 或 MDX 文件放入文件夹，即可生成包含导航、搜索、主题和 Open Graph 图片的生产级文档站点。Blume 在底层驱动一个隐藏的 Astro 项目，核心主题不携带客户端框架 JavaScript，默认获得良好的 Core Web Vitals 评分。它原生支持 AI：在 URL 后追加 .md 可返回原始 Markdown，内置 llms.txt 和 llms-full.txt 文件供 AI 智能体使用，可托管 MCP 服务器供 Claude Code、Cursor 和 VS Code 直接读取文档，并提供可选的页面内 Ask AI 助手。配置为类型安全的 TypeScript 文件，支持 36 种语言和 Notion 等远程内容源。需要 Node.js 22.12 或更新版本。
