# OpenAI 提出管理 AI 投资的五项步骤

## clean_text

OpenAI 提出五项管理 AI 投资的具体步骤，帮助领导者理解使用情况、控制支出并投资于高价值工作。从 GPT-4 到 GPT-5.4，每百万 token 价格下降 97%，GPT-5.6 在 Artificial Analysis Coding Agent Index 中实现更优性能，输出 token 减少 54%，每任务耗时减少 57%。ChatGPT Work 支持更长、多步骤任务，Admin Console 提供按用户、产品和模型的使用分析与支出控制。建议按结果 ROI 评估模型效率，在规模化前治理高级工作流，将 AI 投资作为组合管理，并根据已验证需求匹配容量与支持。
