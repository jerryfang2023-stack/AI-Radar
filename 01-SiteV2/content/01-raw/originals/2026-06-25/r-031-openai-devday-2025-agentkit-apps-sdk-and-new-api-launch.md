OpenAI DevDay 2025: AgentKit Launch, ChatGPT Apps SDK & Major Updates
Updated: October 6, 2025
OpenAI DevDay 2025 has just concluded with game-changing announcements that reshape the AI development landscape.
OpenAI DevDay 2025 has delivered the most significant developer announcements in the company's history. CEO Sam Altman revealed that ChatGPT now serves 800 million weekly active users —doubling from 400 million just eight months ago—while processing over 6 billion tokens per minute through the API. The event introduced AgentKit, a complete platform for agent development, and the ChatGPT Apps SDK, enabling developers to build interactive applications directly within ChatGPT conversations.
These announcements mark a fundamental shift from AI-as-a-service to AI-as-a-platform, positioning OpenAI to directly compete with established workflow automation platforms. With AgentKit's visual builder and ChatGPT's new app ecosystem, OpenAI is creating an integrated development environment that could disrupt the $6.65 billion marketing automation market currently dominated by platforms like Zapier , Make , and n8n .
⚡ Why This Matters Now: The automation and agent-building landscape is shifting dramatically. If you're currently paying for multiple automation tools, AgentKit could consolidate your workflow into a single, more powerful platform with built-in AI intelligence.
ChatGPT Apps SDK: A New Developer Ecosystem
OpenAI unveiled the ChatGPT Apps SDK in preview, enabling developers to build interactive applications directly within ChatGPT conversations. The SDK provides full-stack capabilities including data connections, action triggers, and interactive UI rendering, all built on the Model Context Protocol (MCP) standard.
Launch partners include major platforms like Canva, Figma, Spotify, Zillow, Booking.com, Coursera, and Expedia . These apps integrate seamlessly into conversations, allowing users to invoke functionality by name or through AI-suggested recommendations. For example, users can ask "Figma, turn this sketch into a workable diagram" and the Figma app will respond inline with editable results.
App Discovery and Monetization
The Apps SDK supports multiple discovery methods including direct name invocation, contextual suggestions during conversations, and a forthcoming directory where developers can publish apps for review. Apps that meet higher design and functionality standards will receive prominent featuring and suggestion placements across ChatGPT's 800 million weekly users.
OpenAI announced future support for multiple monetization methods, including the Agentic Commerce Protocol enabling instant checkout directly within ChatGPT. Users with existing subscriptions to third-party services can log in directly from conversations, while new users will soon be able to purchase access without leaving ChatGPT.
Developer Opportunity: This distribution model gives developers immediate access to hundreds of millions of potential users without traditional app store friction. The approach mirrors mobile app ecosystems but with AI-powered contextual discovery, where apps surface precisely when users need them.
AgentKit: Complete Platform for Agent Development
AgentKit represents OpenAI's most comprehensive developer toolkit, designed to take agents from prototype to production with significantly less friction. The platform addresses one of the biggest challenges in AI development: the complexity of orchestration, evaluation loops, tool connections, and UI building that typically prevents agents from reaching production.
For developers currently using workflow automation platforms like Zapier , Make , Lindy , or MindStudio , AgentKit offers a compelling alternative that combines visual workflow building with GPT-5-class reasoning capabilities.
Three Core Components
Agent Builder: A visual drag-and-drop interface for designing agent workflows without starting with code. Built on the Responses API already used by hundreds of thousands of developers, it allows teams to model complex workflows using common patterns extracted from production agent deployments. Developers can wire up specialized agents, add if/else logic nodes, integrate tools, and customize outputs with visual widgets.
ChatKit: Embeddable chat interfaces that developers can brand and customize for their own applications. The component handles the complete chat experience including conversation history, tool invocation display, and custom widget rendering. This eliminates months of frontend development typically required for conversational AI interfaces.
Enhanced Evaluations: Includes trace grading to understand agent decisions step-by-step, datasets for assessing individual agent nodes, and automated prompt optimization. The platform even supports running evaluations on external models directly within the OpenAI Evals platform, enabling comprehensive performance testing across different AI systems.
Real-World Enterprise Implementations
Real-world deployments demonstrate AgentKit's production readiness. Albertsons , operating over 2,000 grocery stores with 37 million weekly shoppers, built an agent using AgentKit that analyzes sales patterns and recommends promotional adjustments. When ice cream sales dropped 32%, the agent automatically analyzed seasonality, historical trends, and external factors to suggest display changes or local advertising.
HubSpot used AgentKit to enhance Breeze, their AI tool serving hundreds of thousands of organizations worldwide. The custom response widget enables Breeze to search knowledge bases, look up location-specific information, and pull policy details to deliver comprehensive answers across multiple data sources.

While platforms like Zapier dominate with 8,000+ integrations and n8n offers self-hosted control, AgentKit combines the ease-of-use of no-code builders with the power of GPT-5-class reasoning models and native access to ChatGPT's massive user base.

Codex Reaches General Availability
OpenAI announced that Codex , its software engineering agent, has moved from research preview to general availability. Powered by the new GPT-5-Codex model specifically trained for agentic coding, Codex now works across IDE, terminal, GitHub, and cloud environments with unified access through ChatGPT accounts.
Since launching GPT-5-Codex in early August 2025, daily messages across Codex have increased 10x. The model has served over 40 trillion tokens , making it one of OpenAI's fastest-growing models. Internally at OpenAI, almost all new code is written by Codex users, with engineers completing 70% more pull requests per week .
Enterprise Features for Development Teams
Slack Integration: Developers can delegate coding tasks or ask questions directly from team channels or threads, bringing Codex into existing collaboration patterns. This addresses feedback from thousands of development teams who wanted Codex accessible within their communication flows.
Codex SDK: Teams can extend and automate Codex within custom workflows, including the ability to programmatically trigger code reviews, refactoring tasks, or automated testing. The SDK was demonstrated live at DevDay, where voice commands integrated with Codex to modify running applications in real-time.
Admin Tools and Reporting: Environment controls, monitoring dashboards, and analytics for enterprise governance. Organizations like Cisco rolled out Codex across entire engineering organizations, achieving 50% faster code reviews and reducing average project timelines from weeks to days.
GPT-5 Pro API Now Available
OpenAI launched GPT-5 Pro in the API, making their most intelligent model available to all developers. Originally released in August 2025, GPT-5 was trained specifically for steering agents and end-to-end coding. The Pro version delivers PhD-level reasoning capabilities across scientific domains with enhanced accuracy for complex tasks.
Leading coding startups including Cursor, Windsurf, and Vercel have adopted GPT-5 to transform how software gets written and shipped in their applications. The model excels at assisting developers with difficult tasks in domains like finance, legal, and healthcare where high accuracy and reasoning depth are critical.
GPT-5 Pro's API availability removes the barrier that kept advanced reasoning capabilities limited to ChatGPT Plus subscribers, enabling developers to integrate the most capable OpenAI model into production applications.
Voice and Video Model Updates
GPT-Realtime-Mini
OpenAI launched GPT-Realtime-Mini as a smaller, cheaper version of the advanced voice model shipped two months ago. Offering the same voice quality and expressiveness as the full model but at 70% lower cost , it makes conversational AI accessible for budget-conscious applications. The model supports MCP server integration, image inputs, and SIP phone calling.
Sora 2 API Preview
The Sora 2 API Preview gives developers access to the model powering stunning video outputs in their own applications. The most significant improvement is controllability—developers can provide detailed instructions while Sora maintains state and delivers stylized, accurate, composed results. The model can expand iPhone footage into cinematic wide shots and generates rich soundscapes with ambient audio and synchronized effects.
