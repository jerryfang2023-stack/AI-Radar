# Manage Copilot coding agent repository access via the API - GitHub Changelog

## clean_text

Organization owners who want to manage Copilot coding agent access at scale can now do so programmatically with the new Copilot coding agent management REST APIs , available in public preview.
With these APIs, you can:
Decide whether Copilot coding agent is available for no repositories, all repositories, or selected repositories owned by your organization.
Add or remove allowed repositories when your organization is set to allow Only selected repositories .
You can also manage these settings through the UI by navigating to your organization&rsquo;s settings and clicking Copilot then Coding agent .
To learn more, see &ldquo;Managing access to GitHub Copilot coding agent&rdquo; in the GitHub Docs.
