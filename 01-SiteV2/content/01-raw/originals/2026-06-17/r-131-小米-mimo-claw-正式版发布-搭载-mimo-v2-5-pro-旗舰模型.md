小米 MiMo Claw 正式版发布，搭载与 OpenClaw 框架官方深度适配的 MiMo-V2.5-Pro 旗舰模型
2026/6/16 21:46:54
来源： IT之家
作者： 归泷
责编： 归泷
评论：
感谢IT之家网友 顺势而为 、 Autumn_Dream 的线索投递！
IT之家 6 月 16 日消息，Xiaomi MiMo Claw 是一款云端轻量化 Claw 类产品，支持低门槛体验 MiMo 模型与 Agent 能力。小米技术官方今日宣布，该产品正式版发布， 搭载与 OpenClaw 框架官方深度适配的 MiMo-V2.5-Pro 旗舰模型 。
据介绍，本次还联动了金山办公生态，推出全链路文档办公提效方案；新增多项 OpenClaw 核心便捷能力；同时升级免费用户权益，单次体验时长 1h → 4h / 天，并面向高频、专业用户上线了 TokenPlan 分层订阅。
IT之家附 Xiaomi MiMo Claw 正式版能力升级如下：
MiMo 旗舰模型赋能
原生协议兼容： MiMo-V2.5-Pro 为 OpenClaw 官方预置模型，原生适配 MCP 工具调用协议，内置技能语义解析能力，无需额外配置提示词工程。
百万级超长上下文： 搭载超长上下文记忆调度架构，适配长链路复杂任务，支持单会话千次以上连续工具调用，有效减少多步骤任务上下文丢失、逻辑断裂问题。
MTP 三层解码架构： 依托 MTP 三层解码技术，在 OpenClaw 标准 Agent 工作流测试中，全链路任务推理吞吐效率提升约 3 倍，实现框架专属性能优化。
联动金山办公
多格式兼容： 支持 Word、Excel、PPT、PDF 四大主流文件格式，超 95% 文档类型，覆盖学习、职场、统计多场景。
全流程闭环服务： 打通 AI 智能生成、高清在线预览、实时在线编辑全链路，无三方平台跳转，一站式办公提效。
轻量化高效使用： 支持自定义指令创作，也可依托内置模板，一键生成标准化文档。
算力效率升级
在 ClawEval 基准测试中，MiMo-V2.5-Pro 模型任务达标率（Pass³）达 63.8%，依托其优异的 Token 利用效率，同等任务效果下，MiMo Claw 的 Token 消耗较其他同类产品降低约 40%-60%。 同时 MiMo Claw 兼容 TokenPlan 各梯度订阅 ，提供强有力算力保障。
云端免部署
MiMo Claw 采用全云端托管模式，无需本地环境配置、不占用本地设备资源。专属个人 Agent 助力全天候在线，支持后台任务续跑、智能自主纠错，减少本地设备故障导致的任务中断问题。
集成多渠道消息对接、文件管理、多任务并行、异常修复等实用功能，有效降低普通用户使用 AI Agent 的操作门槛。
▲ Xiaomi MiMo Claw × 金山办公功能示例 用户权益如下：
免费用户权益升级
创建即可免费使用， 免费用户每日单次体验时长升级至 4 小时
开放全部核心功能，功能配置贴合日常 Agent 使用场景，可满足日常轻度办公使用需求。
过往用户权益兼顾
叠加升级规则： 支持在原有 TokenPlan 套餐基础上自由加购 MiMo Claw 权益 ；套餐升级时，各项权益、费用独立核算，订单生效后统一重置月度 / 年度有效期。
权益取消规则：已叠加 Claw 权益的套餐支持取消服务，平台将按照未使用时间折算剩余价值。
历史活动额度通用：过往 100T 专项活动、订阅调价活动等渠道赠送额度，均可正常使用。
▲ Xiaomi MiMo Claw 免费版网页生成效果 另外，小米还推出了 TokenPlan 分层订阅服务：MiMo Claw 作为独立 Agent 服务，可灵活叠加至 TokenPlan 所有月度、年度套餐（Lite / Standard / Pro / Max）。直接叠加 MiMo Claw 服务开通即可使用，尽可能兼容存量用户权益。同时，MiMo Claw 月度、年度订阅现已开启限时特惠活动（海外订阅服务暂未开放）。
首次特惠： ¥ 14.9 / 月
限时优惠： ¥ 19.9 / 月、¥ 233.8 / 年
