# OpenRouter 发布 Fusion API，多模型协同以一半成本接近 Claude Fable 5 性能

## clean_text

IT之家 6 月 16 日消息，全球最大 AI 大模型 API 聚合平台 OpenRouter 于 6 月 14 日发布公告，宣布推出 Fusion API 复合 AI 模型， 只需约一半成本就能实现 Claude Fable 5 级别性能。
IT之家注，Fusion 是个多模型协同系统服务，会把用户输入同时发给多个模型，再汇总分析各模型结果，最后生成统一回答。
官方测试组合包括 Claude Opus 4.8 与 GPT-5.5、双 Claude Opus 4.8 等“双强”组合，以及 Claude Opus 4.8、GPT-5.5、Gemini 3.1 Pro“三强”组合。

第 1 步是并行发送请求，让多个模型各自作答。
第 2 步是由审查模型读取这些回答，输出结构化分析结果。
第 3 步是由调用模型依据分析内容生成最终答复，实现多模型互补后的综合效果。

成本方面，Gemini 3 Flash＋Kimi K2.6＋DeepSeek V4 Pro 组合 以约 Claude Fable 5 一半的成本，把基准测试分数差距控制在 1％。
