GitHub Introduces Agent HQ
October 28, 2025
GitHub announced Agent HQ — the company's vision for the next evolution of the platform — as it brings coding agents directly within GitHub.
Agent HQ transforms GitHub into an open ecosystem that unites every agent on a single platform. Over the coming months, coding agents from Anthropic, OpenAI, Google, Cognition, xAI, and more will become available directly within GitHub as part of the paid GitHub Copilot subscription.
With Agent HQ, GitHub is ensuring agents aren't bolted on but integrate seamlessly into the ways developers already work, giving them the power to orchestrate their agents to perform tasks in parallel, within the primitives they already know.
To bring its Agent HQ vision to life, GitHub has announced:
- Coding agents from Anthropic, OpenAI, Google, Cognition, and xAI will be available in GitHub as part of a paid Copilot subscription.
- Mission control, a unified command center to see, steer, and approve the work across a fleet of agents in GitHub and VS Code.
- Extended reach with Copilot integrations for Slack and Linear, plus deeper ties into VS Code, including Plan Mode, AGENTS.md files, custom agents, and GitHub MCP Registry access.
- Increased confidence and control across code quality and health with AI controls and enhanced Copilot code review.
