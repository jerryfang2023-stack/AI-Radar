# AWS Partner Guide to AI Agents and Tools in AWS Marketplace

AWS Partner Network (APN) Blog
AWS Partner Guide to AI Agents and Tools in AWS Marketplace
By Vishwa Kurakundi, Partner Solutions Architect – AWS
By Jessica Ho, Sr. Manager, Partner Solutions Architecture – AWS
Generative AI is transforming organizations across all industries. AI Agents represent the next leap in generative AI—evolving from simple chatbots to sophisticated systems capable of complex reasoning and task execution.
As organizations implement these agents to automate specific business processes; from medical research to intelligent enterprise application management and personal assistants, they need trusted expertise to guide their journey. This is where AWS Partners play a critical role, defining how enterprises adopt and govern autonomous systems. From products that offer composable agent toolkits to secure multi-agent collaboration, AWS Marketplace is the central source for this new wave of AI-powered solutions.
Introducing AI Agents and Tools in AWS Marketplace
At AWS Summit New York City, we announced AI Agents and Tools in AWS Marketplace , the go-to destination for businesses seeking to leverage AI agent capabilities. Now, customers can discover pre-built agents and agent tools, as well as professional services, agent development solutions, and software with embedded agents from AWS Partners.
Using natural language search, customers can easily describe their use cases and find relevant Partner solutions. They can choose from pre-built AI agents to automate tasks or deploy agent tools like guardrails, web search tools, and knowledge bases. For more complex needs, customers can engage with Services Partners, leveraging their expertise to build and scale agentic workflows. With hundreds of agent solutions available at launch, AWS Marketplace is the central source for customers to discover, procure, and deploy Partner-built AI agents and agent tools.
For AWS Technology and Services Partners, this launch offers new opportunities to package, list, and scale AI agent solutions and services directly to AWS customers. Leveraging AWS Marketplace provides the following benefits:
Global reach : Tap into the global AWS customer base across industries and geographies.
Enhanced visibility : Showcase AI solutions and services in a dedicated, highly visible category in AWS Marketplace.
Simplified operations : Streamline subscription and billing management.
Flexible deployment and pricing : Choose from new delivery methods and pricing options to match business needs.
Co-sell benefit : The new API-based delivery method is eligible for the SaaS Co-Sell Benefit , enabling incentives to the AWS field for co-selling compliant Partner solutions.
New delivery methods to list AI agents and agent tool solutions
We also announced two new delivery methods in AWS Marketplace that enable Partners to provide a variety of agentic solutions in AWS Marketplace, including those implementing Model Context Protocol (MCP) or Agent-to-Agent (A2A) communication patterns. Additionally, tools like guardrails, knowledge bases, and web search tools help agents operate more accurately and respond with improved agent relevance.
API-based delivery method
This method provides Partners with the ability to expose their vendor-hosted AI agents and tools as APIs, enabling buyers to access these services from anywhere using API keys—without having to provision extensive cloud infrastructure on their end.
Sellers can create free or paid listings with multiple pricing options. The supported pricing models for paid listings include contracts with upfront payment, subscriptions with pay-as-you-go pricing, and a combination of both (contracts with pay-as-you-go pricing). Sellers also have the option to create free trial offers.
Container-based delivery method
This delivery method leverages Amazon Bedrock AgentCore , which provides modular and flexible services for enterprise-ready AI agents. Partners can publish pre-configured docker containers featuring various frameworks like LangChain, LangGraph, and CrewAI, along with foundation models integrated into AgentCore Runtime.
Customers can securely deploy these containerized agents and tools directly into their Amazon Virtual Private Cloud . This approach ensures seamless integration with existing AWS infrastructure, simplifying the deployment and management of AI solutions while maintaining security and scalability.
Understanding AI agents and agent tools
To help Partners effectively categorize their solutions in AWS Marketplace, we’ve established clear definitions for AI agents and agent tools. These classifications will help you position your solutions for maximum visibility and customer relevance.
AI agents are autonomous software applications that independently execute tasks to achieve specific goals. Unlike traditional applications, these agents make decisions and take actions based on collected data. For example, a contact center AI agent can automatically handle customer queries by asking relevant questions, consulting internal documentation, and either resolving issues directly or escalating to human agents when needed.
Agent tools serve as the building blocks that expand an AI agent’s capabilities. These modular components enable agents to perform specialized tasks through external functions, APIs, or services. AWS Marketplace recognizes a number of categories of agent tools, including:
Knowledge bases : These structured information sources support agent queries, such as vector stores, databases, and FAQs. They can include product documentation and customer FAQs.
Guardrails : These tools ensure safe and compliant agent behavior through access controls, response filters, and compliance checks. For example, an HR-focused agent uses guardrails to enforce data privacy protocols and prevent the sharing of confidential employee information.
MCP servers : These handle complex workflows through standardized interfaces, managing API orchestration and business logic. They’re ideal for tasks like compliance validation and multi-step workflow processing.
Best practices for creating an effective AWS Marketplace listing
Follow these best practices to optimize your AI agent and tool listings in AWS Marketplace and maximize customer visibility:
Tag your product appropriately in the AWS Marketplace Management Portal by selecting the AI Agents & Tools category. This helps enhance customer discovery of your solution.
Write clear, specific descriptions of your agent or agent tool’s function (e.g., “data summarization agent for financial analysts”).
Include links to comprehensive documentation and examples to help customers evaluate and experiment with your solution.
For API-based listings, validate architecture detail to qualify for the SaaS Co-Sell Benefit.
Offer a free trial period to allow customers to evaluate your solution. Learn how to create a SaaS free trial offer (available for API-based listings).
Get started with AI agents and tools in AWS Marketplace
AWS Marketplace is transforming how customers discover, distribute, and deploy AI solutions from AWS Partners while ensuring security and reliability through trusted AWS infrastructure.
Next steps:
If you have existing agentic solutions, tag them under the AI Agents & Tools category to enhance discoverability.
If you have a new agentic solution or service, review AWS Marketplace documentation to determine your ideal delivery model.
If you’re new to AWS Marketplace, start your co-sell journey by becoming an AWS Marketplace Seller .
For questions or support, reach out to your Partner team or contact AWS Marketplace support for guidance.
