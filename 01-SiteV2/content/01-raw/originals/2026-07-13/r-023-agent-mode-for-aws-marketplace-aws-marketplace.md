Agent mode for AWS Marketplace
Agent mode is a conversational interface purpose-built for software procurement on AWS Marketplace. It is powered by a set of specialized agents to help you discover, evaluate, and select software solutions through natural conversation. You can describe your needs in plain language and receive personalized recommendations, recommendations, comparisons, and purchase proposals. Agent mode combines the AWS Marketplace product catalog with specialized knowledge from trusted third parties. It also includes real-time web intelligence.
Agent mode uses three specialized AI agents that work together to guide you through your software procurement journey. The primary agents that power agent mode include the following:
Search Agent : Translates your requirements into relevant product recommendations from the AWS Marketplace catalog.
Comparison Agent : Creates side-by-side comparisons of products with reviews, features, and key differentiators.
Evaluation Agent : Generates detailed reports and purchase proposals with insights gathered throughout your discovery process.
These agents operate over a shared working memory and track your evolving needs across interactions.

Step 2: Receive recommendations
Agent mode provides instant product recommendations followed by personalized insights. It learns from your responses to provide increasingly relevant suggestions.
Step 3: Explore and refine
You can continue the conversation to do one ore more of the following:
Ask questions about specific products.
Request comparisons between options.
Modify or add requirements.
Explore different evaluation criteria.
Step 4: Proceed to purchase
Agent mode directs you to the appropriate next step:
View purchase options on product pages.
Request a demo from the seller.
Request a private offer.
Security
Agent mode operates without authentication or authorization requirements. All information you upload or enter is stored without access restrictions and might be visible to unauthorized users. On shared or public computers, your uploaded information might be visible to other users who access the same system. Do not share confidential data, personally identifiable information (PII), or sensitive business information through this feature. Consider the public nature of any data you submit and ensure it doesn't contain proprietary, financial, or personal information that could cause harm if accessed by unauthorized parties.
What agent mode can and cannot do
Agent mode handles
Software discovery and evaluation questions.
Product comparisons and feature analysis.
Requirement clarification and refinement.
Generating purchase proposals and evaluation reports.
Questions about AWS Marketplace products.
Integration and compatibility inquiries.
Agent mode doesn't handle
Out-of-domain questions unrelated to software procurement, such as math problems or weather.
Questions about competitor cloud platforms.
Private pricing information (for products without public pricing).
Direct procurement or purchase transactions (you'll be directed to appropriate AWS Marketplace pages).
Questions about non-AWS Marketplace products.

Security processing
