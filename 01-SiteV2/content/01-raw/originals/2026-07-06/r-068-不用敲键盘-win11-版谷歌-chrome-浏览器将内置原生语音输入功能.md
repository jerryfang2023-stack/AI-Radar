不用敲键盘，Win11 版谷歌 Chrome 浏览器将内置原生语音输入功能
2026/7/5 12:51:57
来源： IT之家
作者： 远洋
责编： 远洋
评论：
感谢IT之家网友 Coje_He 的线索投递！
IT之家 7 月 5 日消息，谷歌 Chrome 浏览器正为 Windows 11 系统开发一项“语音输入”功能，用户只需对着麦克风说话，就能在电脑上录入文字，无需敲击键盘。
IT之家注意到，微软 Windows 早已支持语音输入功能，按下快捷键 Windows+H 即可随时开启语音听写。使用该功能前，用户需要在 Windows 隐私设置中开启在线语音识别。这项功能适用于所有场景，包括各类浏览器，调用时都使用上述键盘快捷键。不过，谷歌在自主研发专属的语音听写输入功能。
该功能目前尚处于早期开发阶段。在 Windows 系统上测试后发现，其操作逻辑和 Edge 类似：先将文本光标定位至输入框内，右键单击，选择“语音输入”。浏览器会先弹出提示框说明功能使用方法，随后出现语音输入悬浮面板，面板内置麦克风控制按钮与完成键。根据实测，选中该功能选项后，界面仅停留在这一状态，暂无其他交互变化。
提示文字说明：“谷歌 Chrome 浏览器将调用你的麦克风采集语音，并自动整理文字填入输入框。”同时提示还标注：“你的语音音频、转写后的文字，以及你当前操作页面的内容与网址，都会发送至谷歌服务器。”
谷歌本可以仅在输入框右键菜单中增设听写选项，直接复用 Windows 自带的语音识别能力，但谷歌显然另有规划。据 Windows Report 网站从 Chromium 代码评审平台 Gerrit 中发现的代码改动记录，除基础语音输入外，谷歌还在开发独立听写悬浮面板、声波动态动画效果，以及语音操控文本编辑指令等配套功能。
谷歌文档等谷歌办公套件产品早已搭载语音输入功能，但本次 Chrome 浏览器新增的语音输入为独立模块，可直接作用于浏览器内所有文本输入框。
待该功能正式面向 Windows、Mac、Linux、ChromeOS 全平台上线后，用户需要手动开启浏览器内的听写实验性标识，才能启用语音输入。
暂无法确认该听写界面与运行逻辑在 Mac 版 Chrome、Linux、ChromeOS 等其他系统上的最终形态，本次测试仅基于 Windows 11 中发现的开发版本功能。谷歌未来向其余平台推送该功能时，相关界面与操作逻辑可能存在差异，且官方尚未公布正式上线时间。
