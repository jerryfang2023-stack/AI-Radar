## 原始摘要 / 采集文本

Google DeepMind 开发者 Ammaar Reshi 使用 Anthropic 的 Claude Code 和 Fable 5，将 2003 年即时战略游戏《命令与征服：将军绝命时刻》移植到 iPhone 和 iPad。游戏原生运行于 ARM64，无需模拟器，战役、遭遇战和"将军挑战"均支持触控操作。首次构建约 40 分钟，随后经数小时调试，两天内耗尽 Claude Max 配额。Reshi 已开源完整源码，但需自备游戏资产（Steam 约 5 美元）。iPad 长时间游玩可能因内存占用过高崩溃，工程日志记录了所有 bug 与修复。
