微软 Win11 26H1 更至 28000.2269：扩展 XBOX 模式、优化文件管理器
2026/6/10 9:55:48
来源： IT之家
作者： 故渊
责编： 故渊
评论：
感谢IT之家网友 吉茵珂絲 、 软媒新友2010825 的线索投递！
IT之家 6 月 10 日消息，在本月（2026 年 6 月）补丁星期二活动日上，微软面向搭载高通骁龙 X2 系列芯片的设备，推送了适用于 Windows 11 26H1 的 KB5095051 更新， 用户安装后版本号升至 Build 28000.2269。
IT之家附上本次更新相关内容如下：
一、新功能
XBOX 模式在 Windows 11 电脑上可用，包括笔记本电脑、台式机和平板电脑。灵感来自 XBOX 游戏机体验，XBOX 模式提供简化的全屏界面，将游戏置于设备中心位置，同时最小化背景干扰。
玩家可以通过 XBOX 应用、游戏栏设置或按 Windows 键 + F11 进入 Xbox 模式。
二、文件资源管理器
扩展了文件资源管理器中可使用的归档格式列表，包括 uu、cpio、xar 和 NuGet 包（nupkg）。
当应用直接打开文件资源管理器的下载和文档等文件夹时，视图和排序偏好设置将被保留。
修复了在打开此电脑或调整暗黑模式时，详细信息窗格大小时可能出现的白色闪烁。
提高了相关 explorer.exe 进程的可靠性，在关闭文件资源管理器窗口后会正确停止。
三、输入
新增功能：在执行某些操作时，用户可以在兼容的输入设备上感受到触觉反馈效果，例如在 PowerPoint 中对齐对象、窗口吸附或调整大小。这些触觉信号可以在设置 > 蓝牙和其他设备 > 鼠标、触摸板或笔 > 触觉信号中开启或关闭。Surface Slim Pen 2、ASUS Pen 3.0 和带有触觉反馈的 MSI Pen 2 支持此功能。随着硬件合作伙伴发布更新，可能还会支持更多兼容设备，包括 Logitech MX Master 4 等特定鼠标。
新增功能：触摸键盘上的语音输入现在看起来更简洁直观。更新后的设计去除了全屏覆盖层，并在语音输入键上直接显示语音输入动画，减少额外的视觉干扰。
新增功能：阿拉伯语 101 传统键盘布局现已可用。用户可以在时间和语言 > 语言和区域设置中为阿拉伯语选择键盘时添加。
提高了在设置 > 蓝牙和其他设备 > Wheel 下设置自定义工具的可靠性。
提高了语音输入中流畅听写设置的持久性。
提升了 emoji 面板（Windows 键 + 句号）的键盘导航可靠性。
提升了使用 ADLaM 键盘时的打字可靠性。
四、共享
新增功能：拖放托盘现更名为放置托盘。其设置现位于设置 > 系统 > 多任务处理（此前为邻近共享）。放置托盘使用更小的预览视图。此改进有助于防止放置托盘无意中打开，并在屏幕顶部附近工作时更容易关闭。
五、任务栏监控智能体
新增功能：Windows 正在任务栏中添加一种新的方式来监控智能体。此体验支持第一方和第三方应用中的智能体，其中 Microsoft 365 Copilot 应用中的 Researcher 作为首批采用者。当 Researcher 处理报告时，Windows 会在任务栏上显示进度，以便用户一目了然地查看更新。将鼠标悬停在 Microsoft 365 Copilot 图标上即可查看实时进度。报告准备好后，Windows 会通知用户。选择通知或图标即可返回应用并查看和使用结果。
六、企业状态漫游（ESR）
新增功能：ESR 现在可以通过组织的 Windows Backup 策略进行管理。这使得 IT 管理员的设置更加简单。
七、基于策略的微软预装应用移除
新增功能：为 Windows 企业和教育版本的“移除默认 Microsoft Store 包”策略添加了对动态应用移除列表的支持。管理员可以通过使用组策略指定其应用包家族名称来移除额外的 MSIX / APPX 打包应用。动态列表目前尚未在 Intune 设置目录中提供。必须使用组策略或自定义 OMA-URI 进行验证。
八、打印
新增功能：在打印设置中添加了一个新图标，用于显示打印机是否支持 Windows 受保护的打印模式。
九、Windows 驱动策略更新
新增功能：通过改变 Windows 内核对第三方驱动的信任方式来提高 Windows 的安全性。移除了跨签名驱动的默认信任，而来自 Windows 硬件兼容性计划（WHCP）的驱动和受信任的旧版驱动白名单仍然允许。Windows 在启用强制执行前会至少检查 100 小时和三次重启的驱动兼容性。强制执行后，一小部分跨签名驱动可能会被阻止。
十、增强批处理文件的安全性和性能
新增功能：管理员和应用控制业务策略作者现在可以更精细地控制系统如何处理批处理文件和命令提示符（CMD）脚本。从本次发布开始，管理员可以启用更安全的批处理文件处理模式。此模式可防止批处理文件在执行过程中更改。
要启用此设置，需要将以下值添加到注册表：
Key： HKEY_LOCAL_MACHINE\Software\Microsoft\Command Processor
值名称：LockBatchFilesWhenInUse
类型：DWORD
要设置的值：0（禁用）或 1（启用）
十一、微软应用商店
减少了从 Microsoft Store 下载和安装应用时出现的意外错误，包括错误代码 0x80070057、0x80240008 和 0x80073d28。
十二、字体
包含对泰语、老挝语、高棉语和龙塔拉文字符的 Leelawadee UI 字体家族的改进，以增强字形序列、位置和渲染。
十三、音频
改进了第三方驱动程序与 midisrv.exe 的兼容性。
十四、任务栏
提升了任务栏系统托盘区域的加载可靠性。
十五、Windows Hello
提升了 Windows Hello Face 的可靠性。
提升了 Windows Hello 指纹在升级过程中的持久性。
十六、存储
提升了查看设置 > 系统 > 存储 > 高级存储设置 > 磁盘和卷时，查看大容量存储信息的性能。
从命令行格式化 FAT32 卷的大小限制从 32GB 提高到 2TB。
十七、交付优化
改进了内存使用，降低了意外使用大量内存的可能性。
十八、显示和图形
改进了支持显示器颜色配置文件选项的持久性和可用性。
十九、Kiosk Mode
简化了在 Microsoft Edge 是允许的应用之一时，Kiosk Mode 中允许的打包应用的配置。
二十、性能
提升了开机自启动应用的性能（设置 > 应用 > 启动中的应用列表应用）。
二十一、可靠性
带来了底层更改，旨在提升 explorer.exe 的可靠性，包括在登录时、与任务栏菜单和任务视图交互时、从文件资源管理器的快速访问中取消固定项目时，以及更多场景。
二十二、其他更新（Normal rollout）
Secure Boot：通过此更新，Windows 质量更新包含额外的设备目标数据，提高了符合自动接收新安全启动证书条件的设备覆盖率。设备只有在展示足够的成功更新信号后才能接收新证书，以保持受控和分阶段的推出。
广告声明：文内含有的对外跳转链接（包括不限于超链接、二维码、口令等形式），用于传递更多信息，节省甄选时间，结果仅供参考，IT之家所有文章均包含本声明。

感谢IT之家网友 吉茵珂絲 、 软媒新友2010825 的线索投递！
IT之家 6 月 10 日消息，在本月（2026 年 6 月）补丁星期二活动日上，微软面向搭载高通骁龙 X2 系列芯片的设备，推送了适用于 Windows 11 26H1 的 KB5095051 更新， 用户安装后版本号升至 Build 28000.2269。
IT之家附上本次更新相关内容如下：
一、新功能
XBOX 模式在 Windows 11 电脑上可用，包括笔记本电脑、台式机和平板电脑。灵感来自 XBOX 游戏机体验，XBOX 模式提供简化的全屏界面，将游戏置于设备中心位置，同时最小化背景干扰。
玩家可以通过 XBOX 应用、游戏栏设置或按 Windows 键 + F11 进入 Xbox 模式。
二、文件资源管理器
扩展了文件资源管理器中可使用的归档格式列表，包括 uu、cpio、xar 和 NuGet 包（nupkg）。
当应用直接打开文件资源管理器的下载和文档等文件夹时，视图和排序偏好设置将被保留。
修复了在打开此电脑或调整暗黑模式时，详细信息窗格大小时可能出现的白色闪烁。
提高了相关 explorer.exe 进程的可靠性，在关闭文件资源管理器窗口后会正确停止。
三、输入
新增功能：在执行某些操作时，用户可以在兼容的输入设备上感受到触觉反馈效果，例如在 PowerPoint 中对齐对象、窗口吸附或调整大小。这些触觉信号可以在设置 > 蓝牙和其他设备 > 鼠标、触摸板或笔 > 触觉信号中开启或关闭。Surface Slim Pen 2、ASUS Pen 3.0 和带有触觉反馈的 MSI Pen 2 支持此功能。随着硬件合作伙伴发布更新，可能还会支持更多兼容设备，包括 Logitech MX Master 4 等特定鼠标。
新增功能：触摸键盘上的语音输入现在看起来更简洁直观。更新后的设计去除了全屏覆盖层，并在语音输入键上直接显示语音输入动画，减少额外的视觉干扰。
新增功能：阿拉伯语 101 传统键盘布局现已可用。用户可以在时间和语言 > 语言和区域设置中为阿拉伯语选择键盘时添加。
提高了在设置 > 蓝牙和其他设备 > Wheel 下设置自定义工具的可靠性。
提高了语音输入中流畅听写设置的持久性。
提升了 emoji 面板（Windows 键 + 句号）的键盘导航可靠性。
提升了使用 ADLaM 键盘时的打字可靠性。
四、共享
新增功能：拖放托盘现更名为放置托盘。其设置现位于设置 > 系统 > 多任务处理（此前为邻近共享）。放置托盘使用更小的预览视图。此改进有助于防止放置托盘无意中打开，并在屏幕顶部附近工作时更容易关闭。
五、任务栏监控智能体
新增功能：Windows 正在任务栏中添加一种新的方式来监控智能体。此体验支持第一方和第三方应用中的智能体，其中 Microsoft 365 Copilot 应用中的 Researcher 作为首批采用者。当 Researcher 处理报告时，Windows 会在任务栏上显示进度，以便用户一目了然地查看更新。将鼠标悬停在 Microsoft 365 Copilot 图标上即可查看实时进度。报告准备好后，Windows 会通知用户。选择通知或图标即可返回应用并查看和使用结果。
六、企业状态漫游（ESR）
新增功能：ESR 现在可以通过组织的 Windows Backup 策略进行管理。这使得 IT 管理员的设置更加简单。
七、基于策略的微软预装应用移除
新增功能：为 Windows 企业和教育版本的“移除默认 Microsoft Store 包”策略添加了对动态应用移除列表的支持。管理员可以通过使用组策略指定其应用包家族名称来移除额外的 MSIX / APPX 打包应用。动态列表目前尚未在 Intune 设置目录中提供。必须使用组策略或自定义 OMA-URI 进行验证。
八、打印
新增功能：在打印设置中添加了一个新图标，用于显示打印机是否支持 Windows 受保护的打印模式。
九、Windows 驱动策略更新
新增功能：通过改变 Windows 内核对第三方驱动的信任方式来提高 Windows 的安全性。移除了跨签名驱动的默认信任，而来自 Windows 硬件兼容性计划（WHCP）的驱动和受信任的旧版驱动白名单仍然允许。Windows 在启用强制执行前会至少检查 100 小时和三次重启的驱动兼容性。强制执行后，一小部分跨签名驱动可能会被阻止。
十、增强批处理文件的安全性和性能
新增功能：管理员和应用控制业务策略作者现在可以更精细地控制系统如何处理批处理文件和命令提示符（CMD）脚本。从本次发布开始，管理员可以启用更安全的批处理文件处理模式。此模式可防止批处理文件在执行过程中更改。
要启用此设置，需要将以下值添加到注册表：
Key： HKEY_LOCAL_MACHINE\Software\Microsoft\Command Processor
值名称：LockBatchFilesWhenInUse
类型：DWORD
要设置的值：0（禁用）或 1（启用）
十一、微软应用商店
减少了从 Microsoft Store 下载和安装应用时出现的意外错误，包括错误代码 0x80070057、0x80240008 和 0x80073d28。
十二、字体
包含对泰语、老挝语、高棉语和龙塔拉文字符的 Leelawadee UI 字体家族的改进，以增强字形序列、位置和渲染。
十三、音频
改进了第三方驱动程序与 midisrv.exe 的兼容性。
十四、任务栏
提升了任务栏系统托盘区域的加载可靠性。
十五、Windows Hello
提升了 Windows Hello Face 的可靠性。
提升了 Windows Hello 指纹在升级过程中的持久性。
十六、存储
提升了查看设置 > 系统 > 存储 > 高级存储设置 > 磁盘和卷时，查看大容量存储信息的性能。
从命令行格式化 FAT32 卷的大小限制从 32GB 提高到 2TB。
十七、交付优化
改进了内存使用，降低了意外使用大量内存的可能性。
十八、显示和图形
改进了支持显示器颜色配置文件选项的持久性和可用性。
十九、Kiosk Mode
简化了在 Microsoft Edge 是允许的应用之一时，Kiosk Mode 中允许的打包应用的配置。
二十、性能
提升了开机自启动应用的性能（设置 > 应用 > 启动中的应用列表应用）。
二十一、可靠性
带来了底层更改，旨在提升 explorer.exe 的可靠性，包括在登录时、与任务栏菜单和任务视图交互时、从文件资源管理器的快速访问中取消固定项目时，以及更多场景。
二十二、其他更新（Normal rollout）
Secure Boot：通过此更新，Windows 质量更新包含额外的设备目标数据，提高了符合自动接收新安全启动证书条件的设备覆盖率。设备只有在展示足够的成功更新信号后才能接收新证书，以保持受控和分阶段的推出。
