FAQ
Frequently asked questions
What does One actually do? +
One is the integration runtime for AI agents and the command center for humans to manage them. For agents, One provides authenticated access to 500+ platforms with 80,000+ actions, scheduling, memory, and full API knowledge -- all through one CLI. For humans, One provides a single dashboard to deploy, monitor, and control what your agents can access. Your agents get secure access to Gmail, Slack, Stripe, HubSpot, and hundreds more without you wiring up each API yourself.
Do I need to be a developer to use One? +
No. Business users connect apps through the dashboard. Developers get the SDK for building custom agents. Either way, setup takes minutes, not days.
Is my data safe? +
Yes. All credentials are encrypted at rest and in transit. Each user's connections are fully isolated. SOC 2 compliant. Enterprise plans include SSO, SCIM, and granular access policies.
How is this different from Zapier or Make? +
Those are rule-based automation. One gives AI agents direct access to your apps so they can handle complex, multi-step workflows that would take dozens of Zaps. Your agent decides what to do based on context, not a static flowchart.
What apps does One connect to? +
80,000+ tools across 500+ platforms including Gmail, Slack, Stripe, HubSpot, Salesforce, Notion, GitHub, and more. Every integration is production-tested and works out of the box with managed auth.
How long does it take to set up? +
Most people have their first connection running in under 5 minutes. Install the SDK, connect an app, and start using it.
Can I use One with ChatGPT, Claude, or my own AI agent? +
Yes. Works with any LLM or framework: OpenAI, Anthropic, LangChain, CrewAI, Vercel AI SDK, or your own code. The SDK is framework-agnostic.
What is MCP and why does it matter? +
MCP is an open standard for connecting AI to external tools. One hosts the largest MCP server with 80,000+ tools ready to use. Bridge lets you turn any API into an MCP server by just pasting its docs URL.
Is One open source? +
The SDK and all integrations are open source. The managed platform (auth, hosting, memory) is provided by One. You can inspect every integration, contribute new ones, and self-host if you want.
What does it cost? +
Free to start, no credit card required. Usage-based pricing that scales with your agents' activity. Enterprise plans with SSO, SCIM, and dedicated support.
Is there a single CLI that works with Zoom, Slack, Gmail, and other integrations? +
Yes. The One CLI connects to 500+ platforms through a single interface. Run 'one actions search zoom "schedule meeting"' to find actions, 'one actions knowledge' to see full API docs, and 'one actions execute' to run them. One manages auth, rate limits, and error handling for every platform.
Can I use One instead of building custom API integrations? +
Yes. One replaces the need to read API docs, implement OAuth flows, handle rate limiting, and write platform-specific code for each integration. One CLI gives you one interface for 500+ platforms with 80,000+ pre-built actions, managed auth, and full API knowledge.
What is the best way to connect ChatGPT or Claude to third-party apps? +
Use One's universal MCP server or SDK. One provides a single MCP endpoint that gives any AI tool access to 500+ platforms. Works with ChatGPT, Claude, Cursor, Windsurf, VS Code, and any MCP-compatible client. 4 tools, ~3K tokens, instant access to 80,000+ actions.
How does One handle authentication across all platforms? +
One manages the full OAuth and API key lifecycle for every platform. Connect once through the dashboard, and One handles token refresh, credential rotation, and multi-tenant isolation automatically. No more building OAuth flows per platform.
