# 别再说行业 Agent 了，先找到每天出错的订单台

行业 Agent 最危险的幻觉，是以为贴上行业名就有了商业价值。

医疗 Agent、保险 Agent、供应链 Agent，听起来都对。但企业真正掏钱时，问的不是它属于哪个行业，而是它能不能接住那个每天出错、每天返工、每天消耗人工的入口。

Choco 的订单台案例值得看，就因为它不漂亮。它处理的是邮件、短信、语音留言、图片、PDF 和手写订单。对食品分销商来说，这不是一个炫技入口，而是每天会影响接单、发货、库存、毛利和客户关系的地方。

## 订单台不是表单，是一堆隐性经验

OpenAI 的客户页面披露，Choco 每年处理超过 880 万份订单，生产环境处理超过 2000 亿个 AI token。OrderAgent 负责把邮件、短信、图片和文档里的订单转成 ERP 可用信息；VoiceAgent 则让客户通过电话自然下单，并支持非营业时间的接单。

这类流程过去不是没人自动化。传统 OCR 可以识别图片和文档，RPA 可以把规则化信息搬进系统，ERP 插件可以把标准订单接进后台。问题在于，食品分销的订单常常不标准。

客户可能说“五个平时那种牛排”，也可能在手写单里写错品牌、单位或规格。一个 SKU 在系统里有正式名称，在客户嘴里却有简称；同样的“一箱”，不同客户可能指不同包装。订单台看似是录入工作，本质上是在把客户习惯、商品目录、配送时间和历史偏好翻译成系统能执行的订单。

这就是 Choco 想越过传统自动化的地方。它不只是做识别，而是把客户历史、商品目录、单位偏好和上下文放进推理过程里。Choco 自己的页面也把关键放在“训练在你的产品和客户数据上”，而不是只说模型能力。

## 真正的验收，不是智能，是少错一单

OpenAI 页面给出的数字很吸引人：人工订单录入最高减少 50%，销售团队在不增加人手的情况下实现 2 倍生产率提升；在可配置自动化门槛下，错误率维持在 1% 到 5% 以下，并支持 24/7 接单。

Choco 自有页面给出的说法更偏销售，但也补了买方会关心的账本：早期用户可让 50% 订单免人工审核，同时保留人工介入；团队可按客户和订单类型决定哪些自动走、哪些留给人工；它还声称多数团队每个销售代表每周可省 3 到 5 小时。

这些数字都来自 Choco / OpenAI 相关页面，不能当成第三方审计。但它们把预算问题说清楚了：企业不是为“AI 懂食品分销”付费，而是为少录一批单、少查一批留言、少一次返工、多接住一段非营业时间订单付费。

买方真正要算的不是一个功能价格，而是一张运营账：人工录入省下多少小时，错单赔付和退换货能少多少，销售是否真的把时间转向客户维护，AI 调用、ERP 接入、数据清洗和人工复核又新增多少成本。

如果这张账算不清，Agent 就只是新包装；算得清，它才可能进预算。

## 这件事最容易被高估的地方，也在订单台

订单台适合 AI，也最容易暴露 AI 的边界。

食品分销有长尾 SKU，有临时替换，有高损耗商品，有当天配送压力。错一箱普通饮料和错一箱海鲜，成本不是一回事。AI 把 95% 的订单处理得更快，剩下 5% 如果集中在高价值、强时效、低容错订单上，运营团队仍然要付出很高的复核成本。

更麻烦的是复制成本。Choco 页面说，OrderAgent 会连接 ERP，并在客户产品目录和历史订单上训练；还提到可以适配 Top Shelf、Sage 或自定义系统。这听起来是能力，也意味着实施负担。每家分销商的商品命名、客户习惯、价格规则、替代品逻辑和 ERP 字段都不一样。如果每进一家企业都要重新清洗目录、映射字段、校正规则和训练团队，这就不是轻产品，而是带有重交付味道的行业软件。

这也是垂直 Agent 真正的分界线。

如果它只是把 OCR、RPA 和客服语音换成大模型外壳，护城河不会太厚；传统 ERP、行业 SaaS、RPA 厂商或客户自建团队都可以追上。如果它能把订单历史、客户习惯、商品目录和人工修正持续沉淀成行业工作层，它才有机会从“自动化工具”变成“订单台系统”。

## 预算不会给行业名，会给错误成本

所以，Choco 这件事不应被读成“食品分销 Agent 成熟了”。更准确的读法是：它给垂直 Agent 找到了一个接近预算的验收场景。

这个场景有几个特征：高频、混乱、可度量、出错有成本，并且客户不愿意为了系统改变自己的下单习惯。订单台符合这些条件。医疗预约、保险理赔、采购查询也可能符合，但前提不是行业名好听，而是有一条类似的脏流程。

对企业买方来说，判断顺序也该变。不要先问要不要上 Agent，先问：哪条流程每天吞掉人工？哪类错误最贵？哪些订单可以自动走？哪些必须人工复核？ERP 接入要花多少？AI 调用成本和错单责任由谁承担？

Choco 案例的价值，在于它把垂直 Agent 从概念拉回了一张账。算不清错单成本，行业 Agent 只是故事；算清了，才可能进预算。
