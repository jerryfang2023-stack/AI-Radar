# 社群情报库

本目录用于沉淀生财有术、AI破局等社群中出现的行业案例、工具技巧、商业机会和资料链接。

归档原则：

- 原始抓取数据仍保留在网页数据文件中；Obsidian 只保存去重后的可读摘要和链接索引。
- 每天生成一篇日报，适合回看当天社群讨论重点。
- 分类索引只保留最新一次运行的高价值条目，避免知识库被低价值重复帖子淹没。
- 真正值得长期跟踪的案例，再升级到正式 Signal Card 或场景候选资产。
