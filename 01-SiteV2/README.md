# 观澜AI｜WaveSight 01-SiteV2

本目录是 V2.2.1 新站工程、内容生产线和长期知识库的当前入口。

## 目录职责

- `site/`：V2.2.1 网站工程。
- `content/`：每日监测、Raw / Pool、卡片、趋势和内参内容生产线。
- `knowledge/`：Obsidian 长期知识库，用于沉淀信号、案例、观点、趋势和来源卡片。

## 使用规则

- 新站代码只进入 `site/`。
- 内容生产文件只进入 `content/`。
- 长期知识卡只进入 `knowledge/`。
- 不把 Raw / Pool / 报告文件平铺到本目录根层。
- 新窗口启动仍以项目根目录 `AGENTS.md` 和 `context/` 为入口。
