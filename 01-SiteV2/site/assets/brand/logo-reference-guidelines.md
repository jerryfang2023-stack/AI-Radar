# 观澜AI｜WaveSight AI 高还原 Logo 说明

本组资产基于用户提供的横版近景参考图制作，目标是尽量还原参考图的气质与结构，而不是重新发散创意。

## 文件

| 文件 | 用途 |
|---|---|
| `logo-wavesight-reference-horizontal.svg` | 高还原横版主 Logo，适合官网导航、品牌页、报告封面 |
| `logo-wavesight-reference-horizontal-reverse.svg` | 深色背景可用的反白横版 Logo |
| `logo-wavesight-reference-symbol.svg` | 只保留左侧图形符号的 symbol |
| `logo-wavesight-horizontal-light.svg` | 同步为正式浅色横版 Logo |
| `logo-wavesight-horizontal-dark.svg` | 同步为正式反白横版 Logo |

## 颜色值

| 名称 | HEX | 用途 |
|---|---:|---|
| 墨海蓝 | `#071827` | 中文字标、英文字标、第一道主澜线 |
| 深澜蓝 | `#0D355C` | 第二道主澜线 |
| 雾蓝灰 | `#6F7F8F` | 第三道浅澜线 |
| 远山灰 | `#A7ADB4` | 低对比辅助水纹 |
| 香槟金 | `#C8A766` | 地平线 / 远方弧线 |
| 背景白 | `#FFFFFF` | 浅色使用场景 |

## 字体建议

- 中文字标：优先使用 `STSong`、`SimSun`、`Noto Serif SC`、`Source Han Serif SC`、`Songti SC`，气质为现代宋体 / 高级衬线。
- 英文字标：优先使用 `Inter`、`IBM Plex Sans`、`Avenir Next`，全部大写为 `WAVESIGHT AI` 并拉开字距。
- 如需完全锁定外观，后续可将字标转为字体授权明确的 outline SVG。

## 安全区建议

- 横版 Logo 四周至少保留中文字标高度的 `1/2`。
- 图形符号单独使用时，四周至少保留符号高度的 `1/4`。
- 不要把 Logo 贴近容器边缘、图片主体或复杂纹理。

## 最小使用尺寸

- 横版主 Logo：网页中宽度不低于 `160px`，报告封面建议不低于 `220px`。
- Symbol：网页中不低于 `28px × 28px`，App / favicon 建议使用 `48px` 或以上源尺寸导出。
- 英文字标在小尺寸下会先失去可读性，小于 `120px` 宽时优先使用 symbol。

## 禁止用法

- 不添加雷达、眼睛、箭头、信号点、山峰、芯片或 AI 图标。
- 不使用蓝紫渐变、霓虹光效、3D 效果或复杂阴影。
- 不改变地平线与澜线的横向舒展结构。
- 不随意压缩、拉伸或改变中英文比例。
