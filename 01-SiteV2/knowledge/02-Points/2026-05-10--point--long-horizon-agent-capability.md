# 更长任务时距正在成为 Agent 能力的新分水岭

## 原始观点

Alex Albert 提到 Claude Mythos Preview 在 METR 时间跨度基准上的表现，提示更长任务周期可能成为新能力分水岭。

## 观澜解读

企业真正买单的不是回答一次很聪明，而是能不能在几十分钟甚至几小时内稳定推进一个流程，并在出错时可恢复、可审计、可接管。

## 观点角色

hypothesis：用于观察长周期 Agent 能力是否会转化为企业采购门槛。

## 可用于哪些判断

- 长任务 Agent。
- Agent 评估。
- 企业流程自动化。

## 不可用于什么

> 基准表现不能直接等同于真实企业流程成功率。
