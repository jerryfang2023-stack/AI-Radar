# HTML artifact 正在替代 Markdown，Agent 输出开始变成可交互工作物

## 原始观点

Thariq 提出“HTML is the new markdown”，并描述自己开始用 Claude Code 生成 HTML 文档与 artifacts。

## 观澜解读

Agent 生成 HTML artifact 的价值不是更花哨，而是让报告、数据、图表和操作入口变成同一个工作物。对观澜 AI 来说，这提示商业内参未来可以从文章演进为可筛选、可展开、可追踪的决策 artifact。

## 观点角色

hypothesis：作为产品形态与内参交互方向的假设输入。

## 可用于哪些判断

- 商业内参产品形态。
- 交互式报告。
- Agent artifact workflow。

## 不可用于什么

> 个人实践不能证明企业团队已经规模化采用。
