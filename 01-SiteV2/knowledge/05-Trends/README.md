# 趋势库

存放多条 Signal 支撑的长期趋势，不收单点新闻。

Risk Trend 可先进入本目录，使用 `trend_type: risk` 或文件名 `risk-*` 标记。暂不单独新增风险目录，等风险资产积累后再评估是否拆分。

入库要求：

- 至少关联 3 条 Signal 或 2 个以上日期。
- 标明趋势阶段。
- 记录推力、阻力、升降级触发器。
- 定期复核。
