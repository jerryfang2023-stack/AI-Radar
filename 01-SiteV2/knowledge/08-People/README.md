# 人物库｜Builders 观点人物页

存放 Builder、Founder、VC、Researcher、Operator。C-2 后，本目录承接前台「关键信号 > Builders 观点」的人物详情页。

入库要求：

- 人物身份。
- title 与组织。
- 主要观点领域。
- 来源链接。
- 关联观点、公司、趋势、信号、机会和商业内参。
- 以人物 / title / 发言时间线为基线，记录观点延续、修正、转向、冲突。
- 观点只作为判断校准，不替代事实主证据。
