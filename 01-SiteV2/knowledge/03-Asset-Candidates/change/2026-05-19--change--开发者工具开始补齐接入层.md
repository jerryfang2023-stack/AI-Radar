# 开发者工具开始补齐接入层

## 明确变化

Anthropic 宣布收购 Stainless。Stainless 长期为 Anthropic 官方 SDK 提供生成能力，并把 API 规范转成 SDK、命令行工具和 MCP server。

## 一句解释

Anthropic 买下的是 Claude 进入外部系统的连接能力。

## 为什么值得看

Agent 要真正进入企业流程，不能只靠一个模型接口。SDK、CLI 和 MCP server 决定它能连接什么系统、由谁维护、出了问题怎么复盘。

## 商业含义

这会先影响企业开发者平台和 IT 团队。以后企业采购 Agent 平台时，除了模型效果，还会追问接入工具、权限治理、日志和维护责任。

## 技术路线 / 方法变化

技术路线的变化在于，Agent 需要稳定连接 API、工具和企业系统。SDK、CLI 和 MCP server 会变成模型平台走进真实流程的基础设施。
