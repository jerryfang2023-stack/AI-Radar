# 微软将默认向符合条件Win11设备自动安装Microsoft 365 Copilot

## 新闻事实

微软计划在2026年6月中旬至7月中旬，向搭载Microsoft 365桌面客户端的合规Windows设备自动安装Microsoft 365 Copilot独立客户端。系统默认开启安装与激活，企业管理员需手动选择"退出自动部署"才能阻止。部署通过Office内置更新程序完成，欧洲经济区设备免于推送。同时，Word、Excel、PowerPoint、Outlook、Teams及网页版Microsoft 365中的Copilot入口也在全...

## 原文要点

- IT之家 6 月 22 日消息，据 Windows Latest 报道，尽管公众强烈反对，且微软此前在强制预装 Microsoft 365 Copilot 一事上看似流露过些许歉意，但如今这家企业又故态复萌。
- 根据用户设备上安装的 Office 版本不同，即便从未主动使用，Microsoft 365 Copilot 相关组件也正悄悄重新侵入用户的办公套件。
- 微软在管理后台的一则更新中悄悄证实：搭载 Microsoft 365 桌面客户端、且符合条件的 Windows 电脑，将自动安装 Microsoft 365 Copilot 应用。
- 微软表示，如果企业管理员不希望旗下设备自动弹出这款智能助手应用，必须手动选择“退出自动部署”。
- 微软计划在 6 月中旬至 7 月中旬期间，向所有装有 Microsoft 365 桌面软件、符合部署条件的 Windows 设备推送该智能助手应用。
