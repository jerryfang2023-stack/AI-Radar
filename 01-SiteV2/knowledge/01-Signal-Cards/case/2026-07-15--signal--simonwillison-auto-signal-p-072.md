# Simon Willison 在 Codex Desktop 中创建自定义宠物"Pedalican"

## 新闻事实

他通过 GPT-5.6 Sol xhigh 模型描述需求，并借助 gpt-image-2 多轮生成精灵素材，最终制作出一只骑自行车的小鹈鹕"Pedalican"，在桌面上弹跳并更新 Codex 任务状态。

## 原文要点

- Simon Willison 在 Codex Desktop 中意外激活了一个类似 Clippy 的动画机器人"宠物"，随后发现可以自定义创建。他通过 GPT-5.6 Sol xhigh 模型描述需求，并借助 gpt-image-2 多轮生成精灵素材，最终制作出一只骑自行车的小鹈鹕"Pedalican"，在桌面上弹跳并更新 Codex 任务状态。关键实现细节已开源，包括 openai/skills 下的 hatch-pet 和 openai/codex 下的 imagegen 两个技能，均采用 Apache 2.0 许可。
