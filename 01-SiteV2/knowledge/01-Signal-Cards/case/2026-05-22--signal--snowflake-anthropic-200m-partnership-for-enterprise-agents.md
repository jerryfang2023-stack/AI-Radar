# Snowflake 与 Anthropic 扩大合作，把 Claude 推向企业 Agent 部署

谁：Snowflake / Anthropic。

证据是什么：来源为 https://www.anthropic.com/news/snowflake-anthropic-expanded-partnership，对应本地证据 R-003，材料已保留全文、哈希和出处。

## 发生了什么

Anthropic 公布与 Snowflake 的多年合作，协议金额为 2 亿美元，并面向 Snowflake 客户推进 AI Agent 部署。

## 为什么值得看

模型进入数据平台后，企业比较的不是单次调用，而是数据边界、销售流程、工程效率和平台采购责任。

## 商业含义

这会影响数据平台客户的 AI 采购路径，也会把模型公司和云数据平台放进同一轮预算讨论。

## 证据边界

材料来自 Anthropic 官方发布，能确认合作范围和内部使用方向；客户成效和外部采用规模仍需案例补充。

## 继续观察

继续看 Snowflake 客户是否公开采用 Claude Agent，以及销售和工程流程指标是否披露。
