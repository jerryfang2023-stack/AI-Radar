# 微软 GitHub Copilot 接入首个开源模型 Kimi K2.7 Code

## 新闻事实

原始来源标题显示：微软 GitHub Copilot 接入首个开源模型 Kimi K2.7 Code。

## 原文要点

- GitHub Copilot 正式接入月之暗面 Kimi 的开源模型 Kimi K2.7 Code，这是其首次接入开源模型。该模型托管于微软 Azure 平台，按量计费，正逐步向 Copilot Pro、Pro+ 及 Max 订阅方案开放，后续将扩展至 Business、Enterprise 版本。相较前代 K2.6，Kimi K2.7 Code 在长上下文编程场景中指令遵循能力显著提升，长程任务性能增强，过度思考倾向改善，平均 to...
- IT之家 7 月 3 日消息，月之暗面 Kimi 今日宣布，GitHub Copilot 正式接入其开源模型 Kimi K2.
- 7 Code，这是 GitHub Copilot 首次接入开源模型 。
- 7 Code 由 GitHub 托管于微软 Azure 平台，采用 按量计费 模式，正逐步向 Copilot Pro、Pro+ 以及 Max 订阅方案开放。
- GitHub 官方表示，此次推送将分批逐步进行，会持续监测该模型的质量与性能。
