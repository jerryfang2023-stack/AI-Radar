# MCP 协议更新：会话 ID 改为无状态，降低大规模部署门槛

## 新闻事实

Model Context Protocol （MCP） 即将迎来重要更新，将服务器端会话 ID 处理方式改为无状态模式，类似普通网站的工作方式。
