# Ithome 发布 AI 基础设施能力，切入模型部署和算力服务

## 新闻事实

英伟达在虚幻嘉年华 2026 宣布 NVIDIA ACE Game Agent SDK 进入 Beta 测试。该轻量级 C/C++ 框架内置 Qwen 3.5 4B 模型用于决策、Chatterbox Turbo 350M 用于语音合成，包含 UE5 插件，可在 8GB 显存 GPU 上本地运行。SDK 提供 Agent API、Chat API 和 RAG API。已有 KRAFTON 在《绝地求生》推出限时 AI 队友 Ally，...

## 原文要点

- IT之家 6 月 18 日消息，在昨日开幕的虚幻嘉年华 2026 期间，英伟达宣布其用于创建和构建 AI 游戏队友的 NVIDIA ACE Game Agent SDK 现已进入 Beta 测试阶段。
- 这套工具允许游戏开发者为搭载 GeForce RTX 显卡的设备构建本地运行的 AI 角色，并包含面向虚幻引擎 5 的插件，覆盖 AI 驱动的自动语音识别、小语言模型和文本转语音组件。
- KRAFTON 近期在《绝地求生：大逃杀》中推出了限时 Beta 测试的 AI 队友 Ally，用于测试和收集玩家对“人机双排模式”的体验反馈。
- IT之家注意到，另一个开发中的案例是 Creative Assembly 在《全面战争：法老》中加入的 AI 顾问。
- 据英伟达官方介绍，NVIDIA ACE Game Agent SDK 被设计为一个轻量级的 C/C++ 智能体框架，内置即用型 Qwen 3.

## 价值描述

这套工具允许游戏开发者为搭载 GeForce RTX 显卡的设备构建本地运行的 AI 角色，并包含面向虚幻引擎 5 的插件，覆盖 AI 驱动的自动语音识别、小语言模型和文本转语音组件。
