# 月之暗面 K2.7 Code 高速版正式登陆 Kimi Code，成为常驻可选模式

## 新闻事实

月之暗面宣布 K2.7 Code 高速版结束 Beta，成为常驻可选模式。

## 原文要点

- 高速版与普通版使用同一模型，输出速度约为普通版的 5-6 倍，常规编程场景下输出速度约 180 Token/s，短上下文场景可达 260 Token/s。
- 高速版在 Coding Plan 中的用量消耗为普通版的 3 倍。
- IT之家 7 月 11 日消息，月之暗面官方昨晚宣布， K2.7 Code 高速版结束 Beta，成为常驻可选模式 。
- 订阅 Allegretto 及以上会员计划的用户，在 Kimi Code CLI 或其他工具中使用 Coding Plan 进行编程时，无需任何申请，可以直接调用 K2.7 Code HighSpeed。

## 价值描述

价格是普通版的两倍：1M tokens 标准输入和输出分别为 13 元和 54 元，缓存命中输入为 2.6 元。
