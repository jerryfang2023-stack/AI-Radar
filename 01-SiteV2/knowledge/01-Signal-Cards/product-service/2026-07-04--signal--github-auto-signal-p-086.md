# Claude Code v2.1.201 发布：Claude Sonnet 5 会话不再使用对话中系统角色进行约束提醒

## 新闻事实

原始来源标题显示：Claude Code v2.1.201 发布：Claude Sonnet 5 会话不再使用对话中系统角色进行约束提醒。

## 原文要点

- Claude Code v2.1.201 版本中，Claude Sonnet 5 的会话不再在对话中途使用 system role 来发送 harness reminders（约束提醒）。此项改动移除了此前用于对话管理的中间系统角色调用。
