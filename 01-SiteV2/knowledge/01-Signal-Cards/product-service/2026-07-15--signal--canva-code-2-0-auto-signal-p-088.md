# Canva Code 2.0 上线：代码生成提速 75%，从提示到发布提速 30%

## 新闻事实

Canva 发布 Canva Code 2.0，一款 AI 氛围编程平台，让不懂代码的用户通过聊天交互直接制作互动网页和小游戏。

## 原文要点

- 新版本新增超过 50 个模板，支持 HTML 导入和实时协作，代码生成速度提升 75%，从提示到发布速度提升 30%。
- IT之家附上相关演示视频如下： 定位方面，Canva Code 2.0 是一款 AI 氛围编程（Vibe Coding）平台，让用户构建并可视化编辑网站、应用及交互体验，最新版本结合创意控制与最新 AI 编程模型，进一步降低交互体验设计门槛。
- 在交互方面，Canva Code 2.0 新增超过 50 个模板，用户可通过输入提示词、选择预设模板或导入 HTML（超文本标记语言）文件启动项目。
- HTML 导入功能支持用户直接导入在其他 AI 平台创建的页面代码，在 Canva 内继续编辑完善。

## 价值描述

IT之家 7 月 15 日消息，Canva 昨日（7 月 14 日）发布公告， 宣布正式上线 Canva Code 2.0 版本，让完全不懂代码的用户通过聊天交互方式，直接做出互动网页、小游戏等。

## 可见原文片段

在编辑环节，Canva Code 2.0 支持实时协作，团队成员可同步处理同一项目。
