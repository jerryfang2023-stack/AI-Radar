# V2 导航与栏目当前决策

## 最终前台导航

```text
今日观察 / 商业信号 / 趋势追踪 / 商业内参
```

## 路由

| 栏目 | 当前路径 | 说明 |
|---|---|---|
| 首页 | `index.html` | 品牌与当日入口 |
| 今日观察 | `daily.html` | 每日 AI 商业行情综述 |
| 今日观察详情 | `daily-detail.html` | 单日完整文章 |
| 商业信号 | `signals.html` | 变化卡、案例卡、Builders 观点 |
| 商业信号详情 | `signal-detail.html` | 单条信号 / 变化详情 |
| 趋势追踪 | `trend-tracking.html` | 深度趋势报告 + 趋势卡 |
| 趋势追踪详情 | `trend-detail.html` | 单篇趋势报告或机会判断 |
| 商业内参 | `brief.html` | 周刊、半月刊、月刊或专题刊物 |

旧路径 旧机会相关路径 已废弃，不再作为 V2 当前路径。

## 降级与后台化

- `机会判断`：不作为一级导航，进入趋势追踪和商业内参中的机会判断段落。
- `The Point`：不作为一级导航，前台表达为 Builders 观点 / 前沿观点 / 观点参照。
- `Trends`：不作为独立旧趋势页，升级为趋势追踪栏目。
- `Scoring / Priority Engine`：后台化。
- `Tags`：用于筛选、搜索、关系网络和资产治理，不做一级栏目。

## 栏目边界

| 栏目 | 核心交付 | 不做什么 |
|---|---|---|
| 今日观察 | 每个老板每天必读的一篇行情综述 | 新闻列表、Raw/Pool 摘要、卡片拼接 |
| 商业信号 | 可追溯的变化卡、案例卡、观点线索 | 公司 PR 列表、观点热榜 |
| 趋势追踪 | 深度研究报告 + 趋势卡 + 机会判断 | 卡片聚合页、旧机会库、执行方案 |
| 商业内参 | 周期刊物和组合判断 | 结构化填充页、每日内容复述 |

## 当前执行要求

- 所有前台可见文案使用当前栏目名。
- 历史报告和归档目录中的旧名称不作为新任务依据。
- 新窗口以 `AGENTS.md`、`agent-workflow/governance/current-context.md` 和本文件为准。
