# V2 VI Brief

状态：ready-for-dispatch  
建议任务：`V2-3 / WSD-20260507-04-v2-vi-design-direction`

## 目标

为 V2.0 建立新的 VI 与视觉系统，让观澜AI从“可用网站”升级为“有识别度的 AI 商业判断品牌”。

## 必须覆盖

- 品牌关键词与反关键词。
- Logo / wordmark / 中英文品牌呈现规则。
- 色彩系统。
- 字体系统。
- 图像与海报规则。
- 数据可视化规则。
- 栏目页、详情页、长文阅读、Signal 卡、Opportunity 卡、Trend 页母版方向。
- 移动端视觉节奏。

## 硬闸门

本任务是 VI / 页面系统前置任务，必须由 UI / UE Design Director 牵头，不进入 Dev。后续任何 V2 页面 Dev 必须先通过本任务或其后续 Design Director closeout。
