# V2 Tag Mapping Rules

V2 uses three tag layers:

| Layer | Purpose | Governance |
|---|---|---|
| `industryTags` / `jobTags` / `workflowTags` | Heat seed aggregation | Can begin as seed tags |
| `formalTags` | Cross-site governed tags | Must come from `tag-taxonomy.md` |
| `candidateTags` | New terms awaiting governance | Must not directly become formal |

## Rules

- Company, product, person, funding round, and mood words are not formal tags.
- One Signal should usually use no more than 6 formal tags.
- One Opportunity should usually use no more than 7 formal tags.
- New industry, job, and workflow terms enter candidate tags first.
- HeatEvidence must include at least one industry tag and at least one job or workflow tag.
- Tags are judgment assets, not casual labels.
