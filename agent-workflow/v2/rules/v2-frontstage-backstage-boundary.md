# V2 Frontstage / Backstage Boundary

V2 frontstage should show judgment, evidence boundary, and readable summaries. It should not expose internal production mechanics.

## Frontstage Can Show

- Daily Brief judgment.
- Front Signal facts and business meaning.
- Opportunity analysis.
- AI Brief issue summaries and member content.
- Heat changes with evidence explanation.
- Point calibration excerpts where useful.

## Backstage Only

- Raw, Pool, Structured interim states.
- Sync status, JSON, script names, field names, recovery operations.
- HeatEvidence calculation internals.
- Source Registry management.
- Priority Engine internals.
- Admin controls.

## V2 Navigation Direction

The current V2 PRD recommends:

```text
Home / Daily Brief / Signals / Opportunities / AI商业内参
```

The Point and Trends are modules/assets, not short-term standalone top-level channels.
