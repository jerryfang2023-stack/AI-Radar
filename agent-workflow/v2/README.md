# WaveSight AI V2.0 工作区

状态：planning  
建立日期：2026-05-07  
owner：`strategy` / `pm` / `workflow`

## 1. 定位

V2.0 不是在 V1.0 上继续堆页面，而是对观澜AI的判断系统做一次系统升级：

- 新 Agent 体系：从长期岗位协作升级为更清晰的采集、判断、编辑、产品、设计、工程和验收链路。
- 新算法：从日常雷达生成升级为多源采集、信号筛选、机会判断、反证检查和趋势沉淀。
- 新 VI：从当前商业情报桌面方向升级为可长期复用的品牌视觉系统。
- 更多内容源与监测规则：从少量新闻与手工整理升级为来源分层、可信度分级、去重和二次搜索。

## 2. 工作区策略

V2.0 仍归属当前 `01-WaveSight` 项目，不另建孤立项目。建议：

- `01-WaveSight`：继续作为主项目和 V1.0 / V2.0 的单一知识库。
- `agent-workflow/v2/`：V2.0 规划、规范、路线图和派发资产。
- 后续进入代码开发时，再创建独立分支或 worktree，例如 `01-WaveSight-v2-lab`。

当前阶段不直接修改 `04-Site` 页面、生产同步脚本或自动化本体。

## 3. V2.0 前置门禁

进入 V2.0 开发前必须先完成：

1. V1.0 baseline 冻结说明。
2. V2.0 产品宪章与非目标。
3. V2 Agent 体系设计。
4. V2 算法与内容源架构。
5. V2 VI / 页面母版方向。
6. V2 技术工作区与迁移方案。
7. 第一批 V2 任务看板和派发单。
