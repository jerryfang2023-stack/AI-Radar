## 追加修正：人物与原文优先

根据用户反馈，The Point 的阅读层级已调整为：

1. 首页、Daily Brief、栏目页、每日集合页和详情页优先展示“谁说的”。
2. 每条 Point 的主阅读内容优先展示原文摘录，观澜评论只作为简短解释。
3. 栏目页每日卡片增加 Top 人物线索、当日数量和 Top10 / 全部观点入口。
4. 每日集合页右侧保留 Top10 排行榜，主栏继续展示 Top10 和全部观点。
5. 详情页标题层级改为：人物为 H1，原文摘录为第二层重点，来源和分数进入辅助信息。

文案与 UE 规范：

- 栏目页只保留一句栏目价值说明：每天追踪一线 AI 创造者的原话，判断哪些观点正在影响产品、技术和商业机会。
- 每日集合页说明控制为一句话，解释阅读顺序：先看谁在说，再看原文说了什么，最后看它对机会判断有什么影响。
- 内页字号层级已明确：人物标题为最大层级，原文摘录为大号引用，评论、摘要、商业含义和边界进入正文详情区。
- 前台继续避免旧版直译、内部流程词和过度承诺表达。

## 追加修正：同人合并、标题克制与英文翻译

根据用户截图反馈，已追加以下规范和页面修正：

6. 详情页顶部已展示原文引用时，正文不再重复出现“原文摘录”区块。
