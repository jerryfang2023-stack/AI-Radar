# 详情页 UI 规范补充｜2026-05-02

## 适用页面

- Daily Brief 详情页
- Signal 详情区
- Trend 详情页
- Opportunity 详情页

## 标题规范

- 英文标签使用短标签：`Daily Brief`、`Signal Detail`、`Trend Detail`、`Opportunity Card`。
- 中文主标题控制在一屏内可读，不使用首页 Hero 级字号。
- 详情页主标题建议字号：`34px - 52px`。
- 标题区统一靠近页面顶部，避免 Opportunity 详情页出现过大顶部空白。

## 正文规范

- 正文建议字号：`15px`。
- 正文行高：`1.68 - 1.75`。
- 小卡片标题建议字号：`18px - 20px`。
- 摘要、说明、证据来源不使用大标题样式。

## 首页 Decision Brief 关联规范

- `Trend Watch` 必须链接到对应 Trend 详情页。
- `Trend Watch` 每条内容必须包含：赛道名、状态/评分/Signals 数、一句趋势观点。
- `Opportunity Watch` 必须链接到对应 Opportunity 详情页。
- `Opportunity Watch` 每条内容必须包含：机会名、评分/证据/关联趋势、一句话机会定义。

## 文案边界

- 可以输出观点，但不替用户下最终经营、投资或合作判断。
- 避免只展示赛道名或机会名，必须提供一条可读的商业含义。
