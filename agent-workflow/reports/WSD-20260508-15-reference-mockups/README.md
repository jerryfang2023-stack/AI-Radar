# WSD-20260508-15 Reference Mockups

本目录是 `V2-SITE-QUALITY-AUTO` 的参考示意图补充。用途不是替代实现代码，而是在诊断后直接给出可讨论的页面方向图，便于结合诊断意见逐页反馈。

## 页面清单

| 页面 | 参考图 | 设计重点 |
|---|---|---|
| 首页 | `home-reference.png` | 商业内参入口、判断语、决策路径、主视觉与今日主判断 |
| 今日要点 | `daily-reference.png` | Daily decision memo、三条判断线、事实 / 影响 / 边界 |
| 关键信号 | `signals-reference.png` | Signal index、状态密度、先看哪条、非新闻列表 |
| 机会解码 | `opportunities-reference.png` | 最新机会报告、客户 / 流程 / 迁移边界、低频报告架 |
| 商业内参 | `brief-reference.png` | 会员内参封面、周期文件感、热力摘要 |
| 今日要点详情 | `daily-detail-reference.png` | 商业备忘录阅读母版、侧栏元信息、核心判断 |
| 信号详情 | `signal-detail-reference.png` | 一句话判断、机会拆解、反向观察、来源与阶段 |
| 机会详情 | `opportunity-detail-reference.png` | 目标客户、商业模式、观察窗口、边界条件 |

## 生成方式

- 模板文件：`reference-mockups.html`
- 截图尺寸：1440 × 1040
- 生成工具：本地 Edge headless
- 视觉方向：商业情报平台 / AI 趋势内参 / 高端商业决策产品
- 参考气质：Apple / Linear / Stripe / PitchBook / Bloomberg / Notion

## 使用建议

建议逐页回复：

- 保留：哪些信息层级、版式或气质应该进入下一轮实现。
- 删除：哪些示意元素过重、过空、过像报告或过像终端。
- 加强：哪里需要更高数据密度、更强商业分析逻辑或更明确决策路径。
