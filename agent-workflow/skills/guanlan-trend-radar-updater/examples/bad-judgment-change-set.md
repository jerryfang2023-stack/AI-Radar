# Bad judgment change set

Reject a projection that:

- labels a category “升温” or assigns a score;
- selects “representative” cases without a factual rule;
- turns missing collection days into “market activity fell to zero”;
- uses builder opinions or community posts as deployment evidence;
- writes “值得关注” or recommends an action;
- loses the Claim, SourceArtifact or original-source link.
