# Bad Top10 Missing Example

```json
{
  "date": "2026-06-12",
  "status": "partial",
  "frontstageCards": [{ "id": "BS-20260612-01" }],
  "frontstageSelection": { "candidate_count": 18, "selection_date": "2026-06-12" }
}
```

Why it fails: public `top10` is missing. Candidate or selection counts do not substitute for the public Top10 contract.
