# Bad Symptom-Only Audit

```yaml
severity: critical
title: "Daily task fails because the rules are too strict"
evidence: "Several files mention gates and V2."
repair: "Delete old code, relax thresholds, and rewrite the workflow."
```

Why it fails: it proves no active path, cites no current contract or exact evidence, mixes unrelated repairs, and assumes lowering gates will fix the owning defect.
