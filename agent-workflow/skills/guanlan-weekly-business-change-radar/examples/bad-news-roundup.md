# Bad Example: News Roundup Disguised as Radar

Anti-patterns to avoid:

## §0 Data Scope (bad)
```markdown
本周收集了大约100多条信息，涵盖各个AI领域的重要动态。
```

Problem: vague counts ("大约"), no data source paths, no boundary statement, no card type breakdown.

## §1 Conclusion (bad)
```markdown
本周AI行业继续快速发展，OpenAI、Anthropic、Google等大厂都有重要发布，垂直行业AI应用也在加速落地。整体来看机遇与挑战并存。
```

Problem: no thesis, no judgment, generic "机遇与挑战并存", doesn't name a specific shift.

## §2 Trend Heatmap (bad)

| 排名 | 趋势 | 说明 |
|------|------|------|
| 1 | AI Agent | 很多公司都在做Agent |

Problem: no velocity tracking (just saying "many companies"), no evidence sources, no prior-week comparison, no judgment.

## §3 Trend Chain (bad)
```markdown
AI Agent正在快速发展，很多公司如Lio、Kestra都获得了融资，这说明Agent是重要趋势。未来Agent会越来越多，创业者应该关注这个方向。
```

Problems:
- Cites companies but not amounts or specific products
- No five-step chain (tech→product→user→business→opportunity)
- Vague prediction ("会越来越多")
- Generic advice ("应该关注")

## §5 Opportunity Card (bad)
```markdown
机会：AI Agent平台
评分：85/100
```

Problems: no structure, no target user, no supply gap analysis, no scoring breakdown — just a name and a number.

## §6 Contrarian View (bad)
```markdown
有人认为AI不会替代人类工作，但我们认为AI确实会改变工作方式。
```

Problems: doesn't name the mainstream narrative being challenged, no counter-evidence, no implication for decision-makers.

## §8 Conclusions (bad)
```markdown
企业应该积极拥抱AI，创业者应该抓住AI机遇。
```

Problems: generic advice applicable to any week, same advice for all roles, no concrete 1-3 month actions.
