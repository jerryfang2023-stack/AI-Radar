# 07-Opportunities — 商业机会库

从日报、项目、内容中提炼的商业机会。

## 目录说明

`07-Opportunities/` 直接存放所有 AI 创业机会卡，不再按子目录拆分。

## 规则

- 每个机会一个文件
- 机会名称不写具体公司名，公司名放在代表案例或来源证据里
- 必须包含：稳定 `opportunity_id`、`slug`、验证状态、证据等级、优先级、来源、标签、复盘时间
- 状态建议：`spotted` → `validated` → `testing` → `active` → `paused` / `closed`
- 来源链接到日报（`01-WaveSight`）、趋势总表（`03-Trends`）或项目复盘（`06-Projects`）
- 云端部署字段规范见：`08-MOC/观澜AI云端部署目录与字段规范.md`
