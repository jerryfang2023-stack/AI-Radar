## 二、逐条评分说明

### 1. 企业桌面端 Agent 执行层（Amazon Quick）

- 总分：22/30
- 判断：观察
- 加分点：
  - AWS 把 Agent 从聊天入口推进到桌面执行入口，分发与连接器价值更清晰。
  - 需求强度高，直接对应企业“检索-生成-执行”闭环的效率痛点。
  - 与企业 Agent 平台、治理和权限生态天然耦合，后续可衍生多层付费模块。
- 扣分点：
  - 仍是 Preview，客户留存、执行稳定性和实际付费意愿尚未公开验证。
  - 微软、Google 和协作平台可能快速内置同类能力，竞争压力高。
- 评分变化：
  相对 Day1 基线是：初始。
  首次纳入评分，当前更像平台级入口信号，仍需等待企业采用数据。

---

### 2. 合规引擎/Agent 治理（Haast）

- 总分：20/30
- 判断：观察
- 加分点：
  - 受监管行业的 AI 落地瓶颈已明确转向合规、审计和控制层。
  - 可复制性较强，行业模板和治理流程有机会标准化。
  - 与 Agent 执行层同步升温，更容易成为“生产化必需件”而不是可选插件。
- 扣分点：
  - 客户规模、上线周期和续费数据仍未公开，商业化深度有限。
  - 云平台和传统安全/GRC 厂商都可能切入，差异化交付是核心难点。
- 评分变化：
  相对 Day1 基线是：初始。
  本次先按“明确方向 + 证据待扩充”处理，后续重点看客户和收入验证。

---

### 3. 国防无人系统任务型基础模型（Scout AI）

- 总分：17/30
- 判断：谨慎
- 加分点：
  - 1 亿美元 A 轮说明资本认可高壁垒任务型 Agent 的长期战略价值。
  - 执行层从软件流程外溢到物理世界，验证 Agent 的边界继续扩大。
  - 仿真、测试、安全和交付工具链存在潜在配套机会。
- 扣分点：
  - 商业化高度依赖合同、资质和项目交付，节奏慢且不可复制性较强。
  - 中国/民用市场迁移路径不清晰，难以直接形成通用创业样板。
- 评分变化：
  相对 Day1 基线是：初始。
  当前更多是资本和赛道外溢信号，不足以支持更积极判断。

---

### 4. 端侧推理芯片 + 软件栈（原粒半导体）

- 总分：17/30
- 判断：谨慎
- 加分点：
  - 融资规模和产业资本质量都强，说明端侧推理窗口被产业链认真对待。
  - 时延、成本和本地合规需求真实，基础需求面并不虚。
  - 一旦形成 SDK 和参考设计生态，长期壁垒可能高于纯应用层。
- 扣分点：
  - 硬件出货、生态绑定和行业适配周期都偏长，回款慢于软件赛道。
  - 纯芯片竞争拥挤，若缺系统级解决方案和渠道能力，商业化会被拖慢。
- 评分变化：
  相对 Day1 基线是：初始。
  当前判断偏保守，等待真实出货、客户绑定和软件栈成熟度数据。

---

### 5. 具身智能“脑”层软件栈（Meta × ARI 并购）

- 总分：16/30
- 判断：谨慎
- 加分点：
  - 大厂并购确认“控制栈/评测/自学习”正在成为高价值软件层。
  - 线下作业场景对稳定执行、安全约束和持续学习的需求真实存在。
  - 对评测平台、数据闭环和任务回放工具形成长期利好。
- 扣分点：
  - 收购金额未披露，直接营收和大规模商用证据不足，短期偏战略动作。
  - 大厂与整机厂自研倾向强，小团队需要避开通用控制栈正面竞争。
- 评分变化：
  相对 Day1 基线是：初始。
  当前仅能确认赛道权重抬升，不能把并购本身等同于成熟商业机会。

---

### 6. 跨境并购与监管变量（Manus 并购受阻）

- 总分：15/30
- 判断：谨慎
- 加分点：
  - 对 Agent 赛道的退出路径、估值方式和合规门槛影响很大。
  - 会提升本地化部署、治理和审计类产品的优先级。
  - 属于高权重外部变量，值得纳入趋势判断而不是忽略。
- 扣分点：
  - 它本身不是可直接复制的商业机会，更像风险信号。
  - 公开细节有限，仍需持续观察后续监管和交易层面的补充信息。
- 评分变化：
  相对 Day1 基线是：初始。
  当前先作为风险约束纳入，不把它当成独立做多方向。

## 四、做多清单

- 企业 Agent 执行层：虽然单条得分仍在观察区间，但 4 月 29 日至 5 月 2 日已连续出现平台级入口、企业级盈利样本和桌面执行入口，重复强证据足以维持做多跟踪。
- AI治理/合规引擎：Vanta 的历史 ARR 验证叠加 Haast 融资与 Manus 并购受阻，说明治理层已从辅助能力升级为企业采购前置条件。

## 五、观察清单

- 企业桌面端 Agent 执行层：方向成立，但需要更多付费客户、留存和执行稳定性验证。
- 合规引擎/Agent 治理：行业痛点清晰，但仍缺公开的部署规模和续费数据。

## 六、回避/谨慎清单

- 国防无人系统任务型基础模型：合同驱动、交付周期长、民用复制路径不清，短期更适合跟踪配套工具链而非正面下注。
- 端侧推理芯片 + 软件栈：产业逻辑成立，但硬件周期和生态壁垒决定其更适合中长期观察。
- 具身智能“脑”层软件栈：大厂并购强化赛道价值，但创业切入点必须避开通用控制栈和整机厂自研。
- 把退出押注在跨境并购的 Agent 资产：监管和地缘变量正在上升，若缺本地合规和营收闭环，风险偏高。

## 八、今日一句判断

当前最值得继续做多跟踪的是“企业 Agent 执行层 + AI治理”，最需要谨慎的是缺乏客户与营收闭环、却把退出押注在跨境并购或大厂收购预期上的 Agent 与具身智能资产。
