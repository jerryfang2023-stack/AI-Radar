企业 AI 的高价值新增证据继续集中在“执行层入口 + 治理层 + 边缘基础设施”，而纯靠资本退出叙事支撑的 Agent 资产不确定性在上升。

### 【机会拆解（6点｜必须详细拆解）】
1. 解决什么具体问题？
   企业知识工作最耗时的环节不是“问不到答案”，而是答案分散在多个系统里，拿到答案后还要执行一串动作。桌面端 Agent 试图把“检索-生成-执行”连成闭环，减少系统跳转和复制粘贴。
2. 目标客户是谁？
   中大型企业的 IT、数字化和业务运营负责人是主要买单方，首批用户会是销售支持、HR、法务、运营和项目协作岗位。
3. 替代或优化什么流程？
   替代手工跨系统查询、整理和回写的链路，优化知识检索、项目跟进、审批提醒、工单流转和周报生成。
4. 商业模式（怎么赚钱）？
   平台更可能采用座席订阅加调用量计费，创业团队更现实的切口是做行业模板、连接器、权限控制和审计模块，按企业年费或项目费收费。
5. 为什么现在值得关注？
   大模型能力差距边际收敛后，分发入口和执行稳定性开始决定商业价值，桌面端是企业 Agent 新一轮分发和治理入口。
6. 是否可迁移中国市场？
   可迁移，但入口更可能是飞书、企微和钉钉；本地化连接器、私有化部署和审计留痕会比通用对话能力更重要。
